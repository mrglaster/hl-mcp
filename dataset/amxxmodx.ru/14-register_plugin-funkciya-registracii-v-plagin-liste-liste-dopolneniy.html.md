# register_plugin Функция регистрации в плагин листе (листе дополнений)
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
**register_plugin** - С помощью данной функции вы сможете передать информацию в плагин лист сервера.  
  
Инфо из **amxmodx.inc:**  
```
/* Sets informations about plugin. Returns the plugin id of the calling plugin. */  
native register_plugin(const plugin_name[],const version[],const author[]);
```

**Синтаксис:**  

register_plugin ( const plugin_name[], const version[], const author[] )
  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <sqlx>  
  
#define PLUGIN "Register plugin"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
}
```
  
л
