# create_kvd
#### Syntax
```
native create_kvd();
```

#### Description
```
Creates a KeyValueData handle.
```

#### Note
```
Handles should be freed using free_kvd().
```

#### Return
```
New KeyValueData handle
```


This code is a part of fakemeta.inc. To use this code you should include fakemeta.inc as ```#include <fakemeta>```
