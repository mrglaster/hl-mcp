# is_str_num Функция проверяет что в строке содержатся только числа
[Ядро AMXx](http://amxxmodx.ru/core/) [string.inc](http://amxxmodx.ru/core/stringinc/)
Функция вернет истину если в строке только числа.Это может пригодится если нужно сделать какой то определенный алгоритм к примеру ввести пароль или еще что то.  
  
**Исходный код функции:**  
```
stock bool:is_str_num(const sString[])  
{  
    new i = 0;  
      
    while (sString[i] && isdigit(sString[i]))  
        ++i;  
      
    return sString[i] == 0 && i != 0;  
}
```
  
  
**Синтаксис:**  

is_str_num(const sString[])
  

    * **const sString[]** строка или массив где и будут происходить действия
  
  
**Тип функции:**  
stock
