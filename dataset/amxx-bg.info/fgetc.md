# fgetc
#### Syntax
```
native fgetc(file);
```

#### Usage
file | ```Handle to the file```
---|---
#### Description
```
Gets character from file.
```

#### Return
```
Character read on success, -1 otherwise
```


This code is a part of file.inc. To use this code you should include file.inc as ```#include <file>```
