# ze_gamemode_set_default
#### Syntax
```
native ze_gamemode_set_default(const game_id);
```

#### Usage
game_id | ```A valid game mode ID.```
---|---
#### Description
```
Description:     Sets a default game mode.
```

#### Return
```
true  | If set successfully.
false | Otherwise.
```

#### Error
```
If game mode ID is not a valid.
```


This code is a part of zombie_escape.inc. To use this code you should include zombie_escape.inc as ```#include <zombie_escape>```
