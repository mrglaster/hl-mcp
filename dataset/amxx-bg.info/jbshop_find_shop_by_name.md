# jbshop_find_shop_by_name
#### Syntax
```
native jbshop_find_shop_by_name(name[])
```

#### Usage
name | ```Shop name```
---|---
#### Description
```
Finds a shop's id by its name.
```

#### Return
```
Shop id on success or JBSHOP_INVALID_VALUE if the shop was not found
```


This code is a part of jbshop.inc. To use this code you should include jbshop.inc as ```#include <jbshop>```


  
  

Warning! This is an external include! It does not come bundled with AMX Mod X "out of the box" and requires additional installation. Use it only if you are absolutely certain that you need it.
