# fm_get_user_maxspeed Функция получает максимальную скорость игрока
[The Fakemeta Module](http://amxxmodx.ru/fakemeta/) [fakemeta_util.inc](http://amxxmodx.ru/fakemeta/fakemeta_utilinc/)
Функция является прямым аналогом функции [**get_user_maxspeed**](http://amxxmodx.ru/fun/funinc/152-get_user_maxspeed-funkciya-vozvraschaet-maksimalnuyu-skorost-igroka.html)  
  
Инфо из **fakemeta_util.inc:**  
```
stock Float:fm_get_user_maxspeed(index) {  
    new Float:speed;  
    pev(index, pev_maxspeed, speed);  
  
    return speed;  
}
```
  
  
**Синтаксис:**  

fm_get_user_maxspeed(index)
  

    * **index** - id игрока  

Функция вернет дробное значение максимальной скорости игрока  
  
**Тип функции:**  
stock  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <fakemeta_util>  
  
#define PLUGIN "[fakemeta_util.inc] fm_get_user_maxspeed"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
   
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
    register_clcmd("say /maxspeed","max_speed")  
   
}  
public max_speed(id){  
       
    new Float:maxspeed = fm_get_user_maxspeed(id)  
    client_print(id,print_chat,"You max speed:%f",maxspeed)  
}
```
