# set_client_listen Функция устанавливает кто кого может слышать
[The Fun Module](http://amxxmodx.ru/fun/) [fun.inc](http://amxxmodx.ru/fun/funinc/)
Представим что вы агент под прикрытием, и вам необходимо подслушать переговоры террористов, с помощью этой функции можно включить, конкретно вам, возможность слушать противника.  
  
Инфо из **fun.inc:**  
```
/* Sets who can listen who. Function returns 0  
* if for some reasons this setting can't be done. */  
native set_client_listen(receiver, sender, listen);
```
  
  
**Синтаксис:**  

set_client_listen ( receiver, sender, listen )
  

    * **receiver** - Кто будет/небудет слышить  

    * **sender** - Кто будет говорить  

    * **listen** - 1 будет слышно / 0 не будет слышно  

  
Функция вернет 0, если по каким то причинам не удастся выполнять операцию.   
  
**Тип функции:**  
Native  
  
**Описание:**  
Так как данная функция не пользуется популярностью, то и не вижу смысла придумывать для нее какой то рабочий пример.  
В крайнем случае вы можете посмотреть описание функции **get_client_listen** , и написать свой собственный рабочий пример.
