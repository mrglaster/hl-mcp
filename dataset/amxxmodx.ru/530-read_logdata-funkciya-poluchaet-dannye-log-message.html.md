# read_logdata Функция получает данные log message.
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Данная функция будет полезна при отладке плагинов, она получает тип log сообщения.  
Обратите внимание что она работает только внутри форварда [**plugin_log**](http://amxxmodx.ru/core/amxmodxinc/524-plugin_log-funkciya-vyzyvaetsya-pri-vseh-log-message-sobytiyah.html)  
  
Инфо из **amxmodx.inc:**  
```
/* Gets log message. Can be called only in plugin_log() forward function. */  
native read_logdata(output[],len);
```
  
  
**Синтаксис:**  

read_logdata(output[],len)
  

    * **output[]** - Массив для данных  

    * **len** - Максимальная длина массива  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <sockets>  
  
#define PLUGIN "[http://amxxmodx.ru] read_logdata"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
      
}  
  
public plugin_log(){  
    new data[256]  
    read_logdata(data,255)  
    server_print("Log message:%s",data)  
      
}
```
  
  
**Описание:**  
После запуска плагина. в консоль сервера будут выводится все log сообщения.
