# dllfunc

The function calls a Half-Life engine DLL function.

```
native dllfunc(type,any:...);
```

type is an DllFunc_* function type, which can be referenced in fakemeta_const.inc 
Make sure the parameters passed are appropriate and match types, otherwise the engine can crash. 


This code is a part of fakemeta.inc. To use this code you should include fakemeta.inc as ```#include <fakemeta>```
