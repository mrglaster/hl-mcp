# get_cvar_string Функция получает значение квара в виде строки
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Благодаря функции **get_cvar_string** мы можем получать значения кваров указанные не в цифрах, а в строках.  
  
Инфо из **amxmodx.inc:**  
```
/* Reads a cvar value. */  
native get_cvar_string(const cvarname[],output[],iLen);
```
  
  
**Синтаксис:**  

get_cvar_string ( const cvarname[],output[],iLen )
  

    * **const cvarname[]** - Имя получаемого квара  

    * **output[]** - Массив для записи полученного значения  

    * **iLen** - Длина массива  

  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
  
#define PLUGIN "Get my cvar String"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
    register_srvcmd("get_my_string_cvar","cvars")  
}  
public cvars(){  
      
    new hostname[64]   
    get_cvar_string("hostname",hostname,63)  
      
    server_print("My cvar hostname = %s",hostname)  
  
}
```
  
  
**Описание:**  
В данном примере мы получим название сервере, где будет установлен данный код. Название сервера хранится в кваре "hostname".  
Выполнив команду **get_my_string_cvar** , начнет выполняться функция **cvars** , в которой мы регистрируем массив под получаемое значение квара и сразу же получаем его, после чего для наглядности выводим в консоль сервера. Команда для выполнения функции разумеется тоже консольная
