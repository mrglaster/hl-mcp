# json_equals
#### Syntax
```
native bool:json_equals(const JSON:value1, const JSON:value2);
```

#### Usage
value1 | ```JSON handle```
---|---
value2 | ```JSON handle```
#### Description
```
Checks if the first value is the same as the second one.
```

#### Return
```
True if they are the same, false otherwise
```

#### Error
```
If passed value is not a valid handle
```


This code is a part of json.inc. To use this code you should include json.inc as ```#include <json>```
