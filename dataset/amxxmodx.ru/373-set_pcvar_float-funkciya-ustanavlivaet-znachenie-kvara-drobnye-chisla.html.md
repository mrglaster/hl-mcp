# set_pcvar_float Функция устанавливает значение квара (дробные числа)
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Функция аналогична функции [**set_pcvar_num**](http://amxxmodx.ru/core/amxmodxinc/372-set_pcvar_num-funkciya-ustanavlivaet-znachenie-kvara-celye-chisla.html), только работает с дробными числами.  
  
Инфо из **amxmodx.inc:**  
```
  
native set_pcvar_float(pcvar, Float:num);
```
  
  
**Синтаксис:**  

set_pcvar_float(pcvar, Float:num)
  

    * **pcvar** - id квара  

    * **Float:num** - Новое значение для квара ( дробные числа)  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
  
#define PLUGIN "[amxmodx.inc] set_pcvar_float"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
new pcvar  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
    //Получение id квара  
    pcvar = register_cvar("test_pcvar","9.8")  
      
    //Переменная для получаемого значения  
    new Float:num  
    //Получаем значение квара  
    num = get_pcvar_float(pcvar)  
      
    //Вывод в консоль севрера полученого значения      
    server_print("^n Pcvar:%.1f ^n",num)  
      
    register_concmd("set_cvar","set_cvar")  
      
}  
public set_cvar(){  
      
    set_pcvar_float(pcvar,5.5)  
      
    new Float:num = get_pcvar_float(pcvar)  
    server_print("^n Pcvar:%.1f ^n",num)  
  
}
```
