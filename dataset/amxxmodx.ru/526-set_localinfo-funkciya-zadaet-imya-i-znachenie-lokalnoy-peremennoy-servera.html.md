# set_localinfo Функция задает имя и значение локальной переменной сервера
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Функция **set_localinfo** устанавливает имя и значение локальной переменной сервера.  
Можно например использовать для передачи типа режима для deathrun  
  
Инфо из **amxmodx.inc:**  
```
/* Sets info for server. */  
native set_localinfo(const info[],const value[]);
```
  
  
**Синтаксис:**  

set_localinfo(const info[],const value[])
  

    * **const info[]** - Имя переменной  

    * **const value[]** - Значение переменной  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <sockets>  
  
#define PLUGIN "[http://amxxmodx.ru] set_localinfo"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
    set_localinfo("test","megatest")  
      
}  
public plugin_log(){  
  
    new array[32]  
    get_localinfo("test",array,31)  
    server_print("LOG MESSAGE %s",array)  
      
      
}
```
  
  
**Описание:**
