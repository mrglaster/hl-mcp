# becomevip_get_flags
#### Syntax
```
native becomevip_get_flags()
```

#### Description
```
Returns the flags that VIP users get as a bitsum.
```

#### Return
```
Flags that VIP users get as a bitsum
```


This code is a part of becomevip.inc. To use this code you should include becomevip.inc as ```#include <becomevip>```


  
  

Warning! This is an external include! It does not come bundled with AMX Mod X "out of the box" and is provided with basebuilder mod.
