Description
set_client_listen - Sets who can listen who. 
Syntax
set_client_listen ( receiver, sender, listen ) 
Type
Native 
Notes
The function returns 0 if for some reasons this setting can't be done.   
Receiver and sender are players indices from 1 to 32.
