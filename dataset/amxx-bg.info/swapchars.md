# swapchars
#### Syntax
```
native swapchars(c);
```

#### Usage
c | ```Value to swap```
---|---
#### Description
```
Swaps the bytes of a value (the lowest byte becomes the highest byte).
```

#### Return
```
Byte-swapped value
```


This code is a part of core.inc. To use this code you should include core.inc as ```#include <core>```
