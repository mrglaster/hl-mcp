# floatstr
#### Syntax
```
native Float:floatstr(const string[]);
```

#### Usage
string | ```Input string to be converted```
---|---
#### Description
```
Converts a string into a floating point value.
```

#### Return
```
Converted value
```


This code is a part of float.inc. To use this code you should include float.inc as ```#include <float>```
