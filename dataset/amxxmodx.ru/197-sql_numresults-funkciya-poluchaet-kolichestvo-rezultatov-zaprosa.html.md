# SQL_NumResults Функция получает количество результатов запроса
[SQLx](http://amxxmodx.ru/sqlx/)
В различных случаях перед обработкой результатов запроса, необходимо знать количество этих самых результатов.  
Функция **SQL_NumResults** получает количество строк полученных в результате выполненного запроса к базе данных.  
  
Инфо из **sqlx.inc:**  
```
/**  
 * Returns the number of rows total.  
 */  
native SQL_NumResults(Handle:query);
```
  
  
**Синтаксис:**  

SQL_NumResults ( Handle:query )
  

    * **Handle:query** - Результат какого запроса хотим обработать.  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <sqlx>  
  
#define PLUGIN "[SQLx.inc] SQL_NumResults"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
new Handle:SQL_Tuple  
new Handle:SQL_Connection   
  
public plugin_init() {  
      
    register_plugin(PLUGIN, VERSION, AUTHOR)  
    register_cvar("SQLx_host","127.0.0.1")  
    register_cvar("SQLx_db","amxxmodx")  
    register_cvar("SQLx_user","admin")  
    register_cvar("SQLx_password","pass")  
      
    register_concmd("select_sql","sqlx_select")  
}  
public plugin_cfg(){  
    new host[33],dbase[33],user[33],pass[33]  
    get_cvar_string("SQLx_host",host,30)  
    get_cvar_string("SQLx_db",dbase,30)  
    get_cvar_string("SQLx_user",user,30)  
    get_cvar_string("SQLx_password",pass,30)  
    SQL_Tuple = SQL_MakeDbTuple(host,user,pass,dbase) // Создание картежа данных для соединения с БД  
    new err, error[256]  
    SQL_Connection = SQL_Connect(SQL_Tuple, err, error, charsmax(error)) // Соединение с БД  
      
    if(SQL_Connection != Empty_Handle)  
    {  
        log_amx("[SQLx connect ok]") //Если соединение установлено   
        }else{  
        log_amx("[SQLX sql error] %s ",error) //Если нет и выводим причину(ошибку)  
        pause("a")  
    }  
      
}  
public sqlx_select(id){  
      
    //Подготовка Запроса  
    new Handle:query = SQL_PrepareQuery(SQL_Connection,"SELECT * FROM `table`")  
    // Выполняем запрос к базе даанных  
    SQL_Execute (query)  
      
    //Объявление новой переменной и получение количества результатов запроса  
    new Sql_result = SQL_NumResults(query)  
    //Вывод результата в консоль сервера  
    server_print("SQL_NumResults:%d",Sql_result)  
      
}  
public plugin_end(){  
  
    SQL_FreeHandle(SQL_Connection)  
  
}
```
  
  
**Описание:**  
Перед использованием примера, не забудьте заполнить все квары и изменить запрос к базе данных под свою таблицу.  
После этого можете протестировать пример, выполнив в консоли команду **select_sql** , в результате вы получите число строк которые подходят под условие в запросе к базе данных.
