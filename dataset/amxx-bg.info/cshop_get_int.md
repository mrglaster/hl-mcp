# cshop_get_int
#### Syntax
```
native cshop_get_int(item, setting[])
```

#### Usage
item | ```Item id.```
---|---
setting | ```Setting's name.```
#### Description
```
Returns an integer value from a specific item's setting.
```

#### Return
```
Setting's value.
```


This code is a part of customshop.inc. To use this code you should include customshop.inc as ```#include <customshop>```
