# nnc_change_user_name
#### Syntax
```
native nnc_change_user_name(id, const name[], bool:showmsg = true)
```

#### Usage
id | ```Client index```
---|---
name | ```Name to set```
showmsg | ```If set to false, the name change message won't display in chat```
#### Description
```
Changes a player's name, bypassing the plugin's protection.
```

#### Return
```
This function has no return value.
```


This code is a part of nnc.inc. To use this code you should include nnc.inc as ```#include <nnc>```
