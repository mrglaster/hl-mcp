# set_fadescreen
#### Syntax
```
stock set_fadescreen(id, iDuration = 0, iHoldTime = 0, iFlags = 0x0000, iRed = 0, iGreen = 0, iBlue = 0, iAlpha = 0)
```

#### Usage
id | ```Client index.```
---|---
iDuration | ```Duration of fade.```
iHoldTime | ```Hold time.```
iFlags | ```Fade type.```
iRed | ```Red color (0..255)```
iGreen | ```Green color (0..255)```
iBlue | ```Blue color (0..255)```
iAlpha | ```Fade brightness (0..255)```
#### Description
```
Set a player fade screen.
```

#### Return
```
This function has no return value.
```
