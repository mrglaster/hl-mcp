od_set_fuse
[DoDFun](http://127.0.0.1:8000/content/index.htm) (dodfun.inc) 
Description
dod_set_fuse - Sets fuses for grenades 
Syntax
dod_set_fuse ( index, set = FUSE_SET, Float:newFuse = 5.0, Type = FT_NEW ) 
Type
Native 
Notes
Valid newFuse number is from 0.1-20.0. Types: new or preprimed (FT_NEW, FT_OLD).
