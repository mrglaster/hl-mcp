# copyc Функция копирует часть строки до определенного символа
[Ядро AMXx](http://amxxmodx.ru/core/) [string.inc](http://amxxmodx.ru/core/stringinc/)
Что бы понять как работает данная функция, я проводил простые эксперименты. Эти эксперименты показали что данная функция копирует левую часть строки до определенного символа.  
  
Инфо из **string.inc:**  
```
/* Copies one string to another until char ch is found.   
*  By len var you may specify max. number of chars to copy. */  
native copyc(dest[],len,const src[],ch);
```
  
  
**Синтаксис:**  

copyc(dest[],len,const src[],ch)
  

    * **dest[]** - Массив куда записывается результат  

    * **len** - Размер массива  

    * **const src[]** - Строка от куда копируется  

    * **ch** - Символ до которого копируется  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
  
#define PLUGIN "[string.inc] copyc"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
    //Новый пустой массив  
    new string[64]  
    //Новый массив(строка) с текстом      
    new string2[] = "Test mega text"  
  
    //Копируем строку до символа g  
    copyc(string,charsmax(string),string2,'g')  
  
    //Вывод обоих строк для наглядности  
    server_print("^n^n%s",string)  
    server_print("%s ^n^n",string2)  
  
}
```
