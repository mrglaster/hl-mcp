# floatfract
#### Syntax
```
native Float:floatfract(Float:value);
```

#### Usage
string | ```Floating point value to get the fractional part from```
---|---
#### Description
```
Returns the fractional part of a floating point value
```

#### Return
```
The fractional part
```


This code is a part of float.inc. To use this code you should include float.inc as ```#include <float>```
