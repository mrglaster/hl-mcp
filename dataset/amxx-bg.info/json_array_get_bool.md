# json_array_get_bool
#### Syntax
```
native bool:json_array_get_bool(const JSON:array, index);
```

#### Usage
array | ```Array handle```
---|---
index | ```Position in the array (starting from 0)```
#### Description
```
Gets a boolean value from the array.
```

#### Return
```
Boolean value
```

#### Error
```
If passed handle is not a valid array
```


This code is a part of json.inc. To use this code you should include json.inc as ```#include <json>```
