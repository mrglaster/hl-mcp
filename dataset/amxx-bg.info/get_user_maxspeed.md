# get_user_maxspeed
#### Syntax
```
native Float:get_user_maxspeed(index);
```

#### Usage
index | ```Client index```
---|---
#### Description
```
Gets player's maximum movement speed.
```

#### Return
```
Player's maximum movement speed
```

#### Error
```
If player is not connected or not within the range
of 1 to MaxClients.
```


This code is a part of fun.inc. To use this code you should include fun.inc as ```#include <fun>```
