# TrieGetSize
#### Syntax
```
native TrieGetSize(Trie:handle);
```

#### Usage
handle | ```Map handle```
---|---
#### Description
```
Returns the number of entries in a hash map.
```

#### Return
```
Number of elements in the hash map
```

#### Error
```
If an invalid handle is provided an error will be
thrown.
```


This code is a part of celltrie.inc. To use this code you should include celltrie.inc as ```#include <celltrie>```
