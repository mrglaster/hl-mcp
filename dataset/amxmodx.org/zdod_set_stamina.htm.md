dod_set_stamina
[DoDX](http://127.0.0.1:8000/content/index.htm) (dodx.inc) 
Description
zdod_set_stamina - DEPRECATED. Sets a player's stamina. 
Syntax
zdod_set_stamina ( player, [ set=STAMINA_SET, minvalue=0, maxvalue=100 ] ) 
Type
Native 
Notes
This native is DEPRECATED and no longer available for use. Use [dod_set_stamina](http://127.0.0.1:8000/dodfun/dod_set_stamina.htm) instead.   
  
Example: for full stamina use set_player_stamina(1,STAMINA_SET,100,100)   
  
Value is from 0 - 100
