# OrpheuCreateStruct
#### Syntax
```
native OrpheuStruct:OrpheuCreateStruct(OrpheuStructType:structType)
```

#### Usage
structType | ```The type of the struct```
---|---
#### Description
```
Creates a struct
```

#### Return
```
A handler to the struct
```


This code is a part of orpheu.inc. To use this code you should include orpheu.inc as ```#include <orpheu>```
