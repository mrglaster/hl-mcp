# get_user_ammo Функция получает количество патронов в оружие и запасе у игрока.
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Получить количество патронов у игрока можно различными способами, функция **get_user_ammo** один из них.  
  
Инфо из **amxmodx.inc:**  
```
/* Gets ammo and clip from current weapon. */  
native get_user_ammo(index,weapon,&clip,&ammo);
```
  
  
**Синтаксис:**  

get_user_ammo(index,weapon,&clip,&ammo)
  

    * **index** - Индекс игрока  

    * **weapon** - Оружие ( смотри CSW_* константы)  

    * **& clip** - Патронов в оружие  

    * **& ammo** - Патронов в запасе  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
  
#define PLUGIN "[http://amxxmodx.ru] get_user_ammo"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
    register_clcmd("say /ammo","get_ammo")  
}  
public get_ammo(id){  
  
    new clip,ammo  
    get_user_ammo(id,CSW_USP,clip,ammo)  
      
    client_print(0,print_chat,"Ammo[%d],clip[%d]",ammo,clip)  
          
}
```
  
  
**Описание:**  
Возьмите пистолет USP и скажите в чат **/ammo** , плагин выведет в чат количество патронв в магазине и запасе.
