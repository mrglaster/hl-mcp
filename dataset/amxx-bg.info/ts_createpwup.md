# ts_createpwup
#### Syntax
```
native ts_createpwup( pwup );
```

#### Description
```
Function will create pwup entity and return its index (pwupent)
```


This code is a part of tsfun.inc. To use this code you should include tsfun.inc as ```#include <tsfun>``` The include is compatible only with The Specialists mod and won't work with other games or mods (Counter Strike 1.6, Half-Life, DoD, etc)
