# set_user_footsteps Функция устанавливает бесшумные шаги игроку
[The Fun Module](http://amxxmodx.ru/fun/) [fun.inc](http://amxxmodx.ru/fun/funinc/)
**set_user_footsteps** - Используя данную функцию, вы сможете сделать бесшумными шаги игрока.  
Этой функцией часто пользуются в плагинах Shop.   
На зомби серверах, для определенных классов и т.п.  
  
Инфо из **fun.inc** :  
```
/* Gives player silent footsteps.  
* if set = 0 it will return footsteps to normal */  
native set_user_footsteps(id, set = 1);
```
  
  
**Синтаксис:**  

set_user_footsteps ( id, [ set = 1 ] )
  
Все предельно просто: 1 - шаги игрока не слышно, 0 шаги снова слышно.  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
   
#include <amxmodx>  
#include <amxmisc>  
#include <fun>  
   
#define PLUGIN "Foot steps"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
   
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
    register_clcmd("say /fsteps","foot_steps")  
}  
   
public foot_steps(id){  
    set_user_footsteps(id,1)  
}
```
  
Как это работает:  
Пишем в чат: **/fsteps**  
И в следующий момент, от вашей ходьбы и бега нет ни звука.
