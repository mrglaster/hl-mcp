# user_silentkill
#### Syntax
```
stock user_silentkill(index, flag = 1)
```

#### Usage
index | ```Client index```
---|---
flag | ```If nonzero, the death will not affect the client's score```
#### Description
```
Kills a user without a message.
```

#### Return
```
This function has no return value.
```


This code is a part of message_stocks.inc. To use this code you should include message_stocks.inc as ```#include <message_stocks>```
