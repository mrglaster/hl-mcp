# client_infochanged Функция вызывается когда игрок изменяет setinfo
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
**client_infochanged** - данная функция вызывается когда игрок изменяет информацию в setinfo. Например изменяет свой никнейм.  
Часто используется в статистиках, в плагинах где есть авторизация, или запрет смены ника и тому подобное.  
  
Инфо из **amxmodx.inc:**  
```
/* Whenever player info is changed, this function is called. */  
forward client_infochanged(id);  

```
  
  
**Синтаксис:**  

public client_infochanged ( id )
  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
  
#define PLUGIN "Changed info"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
}  
public client_infochanged(id){  
    client_print(id,print_chat,"You changed info")  
}
```
  
  
**Описание:**  
Плагин не имеет ни каких команд, данная функция вызывается каждый раз когда игрок меняет свое setinfo.  
Запустив сервер с данным плагином и поменяв к примеру ник, вы в чате увидите сообщение: **You changed info**
