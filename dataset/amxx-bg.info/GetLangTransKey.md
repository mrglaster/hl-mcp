# GetLangTransKey
#### Syntax
```
native TransKey:GetLangTransKey(const key[]);
```

#### Usage
key | ```Key to search for```
---|---
#### Description
```
Finds a translation key index without adding on failure.
```

#### Return
```
Key index, or -1 if not found
```


This code is a part of lang.inc. To use this code you should include lang.inc as ```#include <lang>```
