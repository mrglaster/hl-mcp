s_set_user_tked
[Cstrike](http://127.0.0.1:8000/content/index.htm) (cstrike.inc) 
Description
cs_set_user_tked - Sets whether or not specified user has TKed (team killed). 
Syntax
cs_set_user_tked ( index, [ tk = 1, subtract = 1 ] ) 
Type
Native 
Notes
If tk is 1 then player has tked, otherwise not.   
  
Set subtract to how many frags to subtract. Set subtract to a negative value to add frags.
