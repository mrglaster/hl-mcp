# engfunc Функция вызывает функции движка Half-Life
[The Fakemeta Module](http://amxxmodx.ru/fakemeta/) [fakemeta.inc](http://amxxmodx.ru/fakemeta/fakemetainc/)
Модуль Fakemeta создан для работы с игровыми объектами, в его состав входит функция **engfunc** , которая вызывает функции движка по вашему желанию для конкретного объекта, с ее помощью можно создать или удалить новый объект, и еще много чего другого, полный список возможностей смотрите в файле fakemeta_const.inc **Used with engfunc()**.  
  
Инфо из **fakemeta.inc:**  
```
native engfunc(type,any:...);
```
  
  
**Синтаксис:**  

engfunc ( type, [ ... ] )
  

    * **type** - Тип функции,полный список функций смотри в fakemeta_const.inc Used with engfunc()  

    * **[ ... ]** - Зависит от типа функции  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <fakemeta>  
  
#define PLUGIN "[fakemeta.inc] engfunc"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
      
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
    new Ent = engfunc(EngFunc_CreateEntity)  
    server_print("Entity create? - %s",Ent ? "Yes" : "No")  
  
}
```
  
  
**Описание:**  
Данный пример ни чего визуального не делает. Он создает объект, но этот объект не имеет ни каких свойств, по этому его нельзя увидеть или "потрогать" в игровом мире, можно лишь только проверить создался он или нет, что и выполнено в примере.  
  
В дальнейших материалах по данному модулю будем создавать какой то объект и наделять его различными свойствами и он уже будет виден и вы наглядно сможете посмотреть работу получившегося кода.
