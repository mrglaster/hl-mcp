# register_impulse Регистрация функций для команд impulse *
[The Engine Module](http://amxxmodx.ru/engine/) [engine.inc](http://amxxmodx.ru/engine/engineinc/)
С помощью данной функции вы сможете отловить команды **"impulse *"**.  
Например заблокировать использование спрея (201) или фонарика (100) и другие.  
  
Инфо из **engine.inc:**  
```
/* Registers a client impulse to a function.  Function is passed the ID of the user. */  
native register_impulse(impulse, const function[]);
```
  
  
**Синтаксис:**  

public register_impulse ( impulse, function[] )
  

    * **impulse** - Для какой команды регистрируем 100,101,200  

    * **function[]** - Вызываемая функция  

  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
   
#include <amxmodx>  
#include <amxmisc>  
#include <engine>  
    
#define PLUGIN "Register impulse"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
     
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
  
    register_impulse(100, "fonarik")  
      
}  
public fonarik(id){  
     client_print(id,print_chat,"You use ^"impulse 100^"")  
}
```
  
  
**Описание:**  
В примере отлавливаем событие использование фонарика.
