# cs_set_no_knives Функция устанавливает режим без ножей, отбирает нож у всех
[CStrike](http://amxxmodx.ru/cstrike/)
Функция **cs_set_no_knives** включает/выключает режим "без ножей", но при этом можно выдавать ножи игрокам с помощью других функций.  
Таким образом можно обезоружить всех и дать только одному игроку нож и пускай он попробует догнать остальных и перерезать. Такой мод уже существует и называется Hide'n'Seek, но там это реализована с помощью другой функции и другого модуля.   
  
Инфо из **cstrike.inc:**  
```
/* Enabled no knives mode by calling this with value 1. Disabled with 0.  
 * No knives mode means that player will not be given a knife when spawning.  
 * You can still give knives (ie through fun's give_item).  
 */  
native cs_set_no_knives(noknives = 0);
```
  
  
**Синтаксис:**  

cs_set_no_knives ( [ noknives = 0 ] )
  

    * **[ noknives = 0 ]** - По умолчанию 0 ( ножи вклчюены) 1 - режим без ножей  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <cstrike>  
  
#define PLUGIN "[cstrike.inc] cs_set_no_knives"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
  
    register_clcmd("say /noknive","del_kinve")  
  
}  
  
public del_kinve(id){  
  
    cs_set_no_knives(1)  
}  

```
  
  
**Описание:**  
Хотите включить режим без ножей? Просто наберите в чате команду **/noknife** , которая выключит ножи у всех.  
Обратите внимание, что cs_set_no_knives(**1**) - отбирает, а **0** наоборот дает ножи.
