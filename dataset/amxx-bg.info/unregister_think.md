# unregister_think
#### Syntax
```
native unregister_think(registerid);
```

#### Usage
registerid | ```Think forward id```
---|---
#### Description
```
Removes a previously registered think hook.
```

#### Return
```
1 on success, 0 if nothing was removed
```


This code is a part of engine.inc. To use this code you should include engine.inc as ```#include <engine>```
