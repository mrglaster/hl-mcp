# get_user_gravity
#### Syntax
```
native Float:get_user_gravity(index);
```

#### Usage
index | ```Client index```
---|---
#### Description
```
Gets player's gravity.
```

#### Return
```
Player's gravity value, 1.0 being normal gravity (800)
```

#### Error
```
If player is not connected or not within the range
of 1 to MaxClients.
```


This code is a part of fun.inc. To use this code you should include fun.inc as ```#include <fun>```
