```
stock xs_plane_copy(const Float:planeIn[], Float:planeOut[])
```

#### Usage
planeIn | ```The plane to copy.```
---|---
planeOut | ```The plane to store the copy into.```

```
Copies a plane.
```

#### Return

```
This function has no return value.
```
