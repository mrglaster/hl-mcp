# is_user_hltv Функция проверяет HLTV или игрок
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Некоторые функции используются над игроками, только игроками или наоборот только ботами или HLTV, по этому необходимо узнать кем является проверяемый. Например глупо будет проверять на пинг ботов или давать броню HLTV  
  
Инфо из **amxmodx.inc:**  
```
/* Returns 1 or 0. */  
native is_user_hltv(index);
```
  
  
**Синтаксис:**  

is_user_hltv ( index )
  

    * **index** - id првоеряемого  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
  
#define PLUGIN "[amxmodx.inc] is_user_hltv"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
  
}  
public client_connect(id){  
      
    new name[32]  
    get_user_name(id,name,31)  
      
    if(is_user_hltv(id)){  
          
        server_print("%s is HLTV",name)  
          
          
    }else{  
        server_print("%s is Human",name)  
      
    }  
}
```
  
  
**Описание:**  
В данном примере осуществляется проверка клиента заходящего на сервер, игрок HLTV это или нет.  
При этом клиент подсоединяющийся может быть не только человеком, но и ботом.  
  
Функция в результате своей работы возвращает 1 если это HLTV и 0 если нет.  
  
В коде создано условие если это HLTV то выполнять один код, если нет то другой.  
В результате работы будет выдано текстовое сообщение в консоль сервера с именем присоединившегося клиента.
