# nvault_pset Функция записывает значение по ключу не изменяя дату.
[The nVault Module](http://amxxmodx.ru/nvaultinc/)
Иногда надо изменить данные, но при этом не изменять дату их создания/изменения, для таких нужд применяется функция **nvault_pset**  
  
Инфо из **nvault.inc:**  
```
/* Sets a permanent vault value (no timestamp) */  
native nvault_pset(vault, const key[], const value[]);
```
  
  
**Синтаксис:**  

nvault_pset ( vault, const key[], const value[] )
  

    * **vault** - Идентификатор открытого файла  

    * **const key[]** - Ключ для записи  

    * **const value[]** - Данные для записи  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <nvault>  
  
#define PLUGIN "[nvault.inc] nvault_pset"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
    new hvault = nvault_open("mega_file")  
    new s_ID[] = "zero"  
    nvault_pset(hvault, s_ID, "hello http://amxxmodx.ru")  
      
    new text[64],timestamp  
      
    nvault_lookup(hvault,s_ID,text,charsmax(text),timestamp)  
      
    server_print("[nVault] %s,%d",text,timestamp)  
      
}
```
  
  
**Описание:**  
Функция практически идентична [**nvault_set**](http://amxxmodx.ru/nvaultinc/403-nvault_set-funkciya-ustanavlivaet-znachenie-s-izmeneniem-vremennoy-metki.html), по этому нет смысла повторяться.
