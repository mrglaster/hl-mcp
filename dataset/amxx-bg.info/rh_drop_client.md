# rh_drop_client
#### Syntax
```
native rh_drop_client(const index, const message[] = "");
```

#### Usage
index | ```Client index```
---|---
message | ```Message that will be sent to client when it is deleted from server```
#### Description
```
Kicks a client from server with message
```

#### Return
```
This function has no return value.
```
