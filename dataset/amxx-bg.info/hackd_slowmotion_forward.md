# hackd_slowmotion_forward
#### Syntax
```
native hackd_slowmotion_forward(const handler[]);
```

#### Usage
handler | ```- The forward to call. Signature: public slowmotion_detected(id, Float:percentage, cmds, drops).```
---|---
#### Description
```
Register forward that will be called when slowmotion detected on the client.
```


This code is a part of hackd.inc. To use this code you should include hackd.inc as ```#include <hackd>```


  
  

Warning! This is an external include! It does not come bundled with AMX Mod X "out of the box" and requires additional installation. Use it only if you are absolutely certain that you need it.
