# xs_rad2gra
#### Syntax
```
stock Float:xs_rad2gra(Float:x)
```

#### Usage
x | ```Input radians.```
---|---
#### Description
```
Converts radians to gradians.
```

#### Return
```
Radians converted to gradians.
```


This code is a part of xs.inc. To use this code you should include xs.inc as ```#include <xs>```
