Description
trim - Strips spaces from the beginning and end of a string. 
Syntax
trim ( text[] ) 
Type
Native 
Notes
Returns the number of deleted chars.   
  
**Spaces are:**   
- ordinary spaces   
- tabs   
- new lines   
- carriage returns   
- vertical tabs
