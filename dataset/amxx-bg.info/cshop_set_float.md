# cshop_set_float
#### Syntax
```
native cshop_set_float(item, setting[], Float:value)
```

#### Usage
item | ```Item id.```
---|---
setting | ```Setting's name.```
value | ```Setting's default value.```
#### Description
```
Registers a float setting for a specific item.
```

#### Return
```
This function has no return value.
```


This code is a part of customshop.inc. To use this code you should include customshop.inc as ```#include <customshop>```
