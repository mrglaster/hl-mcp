# get_user_maxspeed Функция возвращает максимальную скорость игрока
[The Fun Module](http://amxxmodx.ru/fun/) [fun.inc](http://amxxmodx.ru/fun/funinc/)
Ни для кого не секрет, что с разным оружием максимальная скорость передвижения изменяется, функция **get_user_maxspeed** расскажет вам какая максимальная скорость доступна игроку в данный момент.  
  
Инфо из **fun.inc:**  
```
/* Returns users max. speed. */  
native Float:get_user_maxspeed(index);
```
  
  
**Синтаксис:**  

Float: get_user_maxspeed ( index )
  

    * **index** - id игрока  

  
Обратите внимание, функция возвращает дробное число.  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <fun>  
  
#define PLUGIN "[fun.inc] get_user_maxspeed"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
    register_clcmd("say /maxspeed","max_speed")  
  
}  
public max_speed(id){  
      
    new Float:maxspeed = get_user_maxspeed(id)  
    client_print(id,print_chat,"You max speed:%f",maxspeed)  
}  

```
  
  
**Описание:**  
Выполните команду **say /maxspeed** , и получите информацию в чат о максимальной скорости, купите более тяжелое оружие чем пистоле и снова выполните данную команду, результат измениться.  
Не забудьте что функция возвращает число с плавающей точкой, по этому переменная куда сохраняем значение должна быть тоже дробной.  
И вывод результатов тоже дробный.
