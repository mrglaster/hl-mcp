# fclose Функция для закрытия файла по идентификатору
[Ядро AMXx](http://amxxmodx.ru/core/) [file.inc](http://amxxmodx.ru/core/fileinc/)
Думаю ни кому не нужно объяснять зачем следует закрывать файлы, все и так хорошо знакомы с операционными системами и знают что будет если не закрывать открытые файлы.  
  
Инфо из **file.inc:**  
```
//Closes a file handle  
native fclose(file);
```
  
  
**Синтаксис:**  

fclose(file)
  

    * **file** - Идентификатор файла который необходимо закрыть.  

  
  
**Тип функции:**  
Native  
  
**Описание:**  
Действие функции носит технический характер, по этом писать отдельный пример для нее не имеет смысла.  
Где берется идентификатор файла и как данная функция употребляется вы можете посмотреть в описании функции [**fopen**](http://amxxmodx.ru/core/fileinc/297-fopen-funkciya-otkrytiya-fayla-i-vozvrata-ego-diskriptora.html)
