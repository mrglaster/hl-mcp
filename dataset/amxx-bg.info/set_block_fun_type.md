# set_block_fun_type
#### Syntax
```
native set_block_fun_type(ent, funtype)
```

#### Usage
ent | ```Индекс на ентитито на блока```
---|---
funtype | ```Забавен тип на блока (BLOCKS_YELLOW - бързодвижещ се блок , BLOCKS_BLACK - недвижим блок , BLOCKS_BLUE - забавящ блок , BLOCKS_RED - запалващ блок)```
#### Description
```
Задава забавен тип на блок
```

#### Return
```
This function has no return value.
```
