# RadiusDamage
#### Syntax
```
stock RadiusDamage(const Float:fExplodeAt[3], iDamageMultiplier, iRadiusMultiplier)
```

#### Description
```
Hurts/Kills players in a sphere, like an explosion, Multiplier determines damage.
```
