# rmdir Функция удаляет каталог (папку).
[Ядро AMXx](http://amxxmodx.ru/core/) [file.inc](http://amxxmodx.ru/core/fileinc/)
Функция необходима для удаления каталога, но помните, что возможно удалить только пустую папку!  
  
Инфо из **file.inc:**  
```
//Attempts to remove a directory.  
//Note that you cannot remove a directory that has files on most  
// operating systems.  
native rmdir(const path[]);
```
  
  
**Синтаксис:**  

rmdir(const path[])
  

    * **const path[]** - Имя папки и путь к ней  

Функция вернет 1 если папка успешно удалена, в других случаях вернет 0  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
  
#define PLUGIN "[http://amxxmodx.ru] rmdir"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
    rmdir("addons/amxmodx/configs/test")  
}
```
  
  
**Описание:**  
Создайте в папке с конфигами папку test, и запустите сервер, папка будет удалена.  
Можете внутри папки попробовать создать любой файл и тогда папка удалена не будет.
