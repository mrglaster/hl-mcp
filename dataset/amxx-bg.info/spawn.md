# spawn
#### Syntax
```
native spawn(index);
```

#### Usage
index | ```Entity index```
---|---
#### Description
```
Spawns an entity.
```

#### Return
```
This function has no return value.
```

#### Error
```
If player is not connected or not within the range
of 1 to MaxClients.
```
