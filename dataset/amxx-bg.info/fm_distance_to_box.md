# fm_distance_to_box
#### Syntax
```
stock Float:fm_distance_to_box(const Float:point[3], const Float:mins[3], const Float:maxs[3]) {
```

#### Description

Finds distance from point to box defined with min origins and max origins


This code is a part of fakemeta_util.inc. To use this code you should include fakemeta_util.inc as ```#include <fakemeta_util>```
