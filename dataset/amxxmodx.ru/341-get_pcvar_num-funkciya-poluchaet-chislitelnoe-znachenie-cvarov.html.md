# get_pcvar_num Функция получает числительное значение cvar`ов
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Функция аналогична функции [**get_pcvar_string**](http://amxxmodx.ru/engine/go.php?url=aHR0cDovL2FteHhtb2R4LnJ1L2NvcmUvYW14bW9keGluYy8zMzUtZ2V0X3BjdmFyX3N0cmluZy1mdW5rY2l5YS1wb2x1Y2hhZXQtc2ltdm9sbm9lLXpuYWNoZW5pZS1jdmFyLmh0bWw%3D), разница лишь в типе получаемых данных, **get_pcvar_num** получает значение квара в виде числа.  
  
Инфо из **amxmodx.inc:**  
```
native get_pcvar_num(pcvar);
```
  
  
**Синтаксис:**  
```
get_pcvar_num(pcvar)
```
  

    * **pcvar** - указатель на cvar, предварительно полученный с помощью функций: register_cvar или get_cvar_pointer.  

Функция возвращает значение указанного квара.  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
  
#define PLUGIN "[amxmodx.inc] get_pcvar_num"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
    //Регистрация квара и получение его id  
    new pcvar = register_cvar("test_pcvar","777")  
      
    //Переменная для получаемого значения  
    new num  
    //Получаем значение квара  
    num = get_pcvar_num(pcvar)  
      
    //Вывод в консоль севрера полученого значения      
    server_print("^n Pcvar:%d ^n",num)  
      
}
```
  
  
**Описание:**  
Основное отличие функции **get_pcvar_num** от **get_pcvar_string** в том, что функция возвращает значение квара, а не записывает в массив.  
Остальное в примере предельно просто и понятно показано.
