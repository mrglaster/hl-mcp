[Core](http://127.0.0.1:8000/content/index.htm) (amxmodx.inc) 
Description
set_user_deaths - Sets a player's death count. 
Syntax
set_user_deaths ( index, newDeaths ) 
Type
Native 
Notes
This does not work on some mods, which store deaths in a private variable, like Counter-Strike.   
  
See [cs_set_user_deaths](http://127.0.0.1:8000/cstrike/cs_set_user_deaths.htm).
