s_set_fov
[NS](http://127.0.0.1:8000/content/index.htm) (ns.inc) 
Description
ns_set_fov - Sets the players field of view. 
Syntax
ns_set_fov ( index, [ Float:fov = 0.0 ] ) 
Type
Native 
Notes
Set fov to 0.0 (or omit it) to return to normal. FOV change will persist until disconnect unless reset by a plugin.
