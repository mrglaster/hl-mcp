# csgor_get_user_skins
#### Syntax
```
native csgor_get_user_skins(id, skinid);
```

#### Usage
id | ```Player index.```
---|---
skinid | ```A valid Skin ID.```
#### Description
```
Returns a player's amount of skins by specified Skin ID.
```

#### Return
```
Amount of skins. -1 on error.
```


This code is a part of csgo_remake.inc. To use this code you should include csgo_remake.inc as ```#include <csgo_remake>```
