s_popup
[NS](http://127.0.0.1:8000/content/index.htm) (ns.inc) 
Description
ns_popup - Sends a popup to a player. 
Syntax
ns_popup ( index, szMsg[180], ah ) 
Type
Native 
Notes
Set index to 0 to send to everybody. The last parameter, if set to 1, will only display when the player has cl_autohelp set to 1.
