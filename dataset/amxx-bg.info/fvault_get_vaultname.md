# fvault_get_vaultname
#### Syntax
```
stock fvault_get_vaultname(const vaultnum, vaultname[], len)
```

#### Usage
vaultnum | ```Vault number to find the vault name```
---|---
vaultname | ```String which vault name will be copied to```
len | ```Length of vault name```
#### Description
```
Retrieves a vault name specified by its number
```

#### Return
```
Returns 1 on success, 0 on failue.
```


This code is a part of fvault.inc. To use this code you should include fvault.inc as ```#include <fvault>```


  
  

Warning! This is an external include! It does not come bundled with AMX Mod X "out of the box" and requires additional installation. Use it only if you are absolutely certain that you need it.
