# Roomtype Reset
[Готовые скрипты](http://amxxmodx.ru/scripts/)
![Roomtype Reset](http://www.17buddies.net/17b2/Btn/4/0/0/0/0/en/191.png)
  
На картах as_tundra, cs_docks, cs_siege, de_luxor и некоторых других есть зоны, где звуки приобретают несколько иное звучание (появляется 'эхо'). Из-за недоработки в клиенте после смены карты у игрока звуки с эффектами могут остаться; лечится это дело рестартом игры или установкой квара _room_type_ в 0. Плагин - обход проблемы: без слоухакинга заставляет зашедших игроков сбросить квар на значение по умолчанию.  
  
Плагин должен быть прописан непосредственно в _plugins.ini_ : когда фикс не нужен - он и не применяется; позиция не важна.
