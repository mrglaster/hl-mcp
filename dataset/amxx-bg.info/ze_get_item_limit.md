# ze_get_item_limit
#### Syntax
```
native ze_get_item_limit(iItemid);
```

#### Usage
iItemid | ```Item id.```
---|---
#### Description
```
Description:     Return item limit.
```

#### Return
```
Limit         | If this itemid is valid.
ZE_WRONG_ITEM | If this itemid is invalid.
```

#### Note
```
ZE_WRONG_ITEM is defined as -1
```


This code is a part of zombie_escape.inc. To use this code you should include zombie_escape.inc as ```#include <zombie_escape>```
