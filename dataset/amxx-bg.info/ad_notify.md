# ad_notify
#### Syntax
```
forward ad_notify(const index, const PunishType:pType, const NotifyType:nType, const Kills, const Shots, const Warn);
```

#### Usage
index | ```Client index```
---|---
pType | ```Punish type```
nType | ```Notify type```
Kills | ```The amount of kills a enemies```
Shots | ```The amount of clean shots```
Warn | ```The current amount of warnings for player```
#### Description
```
Notifies about warnings the player, also detection aim.
```

#### Return
```
This forward ignores the return value.
```


This code is a part of reaimdetector.inc. To use this code you should include reaimdetector.inc as ```#include <reaimdetector>```
