# csgor_get_user_statt_skins
#### Syntax
```
native csgor_get_user_statt_skins(id, skinID);
```

#### Usage
id | ```Player index.```
---|---
skinid | ```Skin ID.```
#### Description
```
Returns amount of StatTrack Skins on certain SkinID.
```

#### Return
```
Amount of StatTrack Skins Num.
```


This code is a part of csgo_remake.inc. To use this code you should include csgo_remake.inc as ```#include <csgo_remake>```
