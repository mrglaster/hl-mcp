# client_disconnect Функция вызыватся когда игрок уходит с сервера
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
**client_disconnect** - данная функция вызывается при отключении игрока с сервера. Используется во многих плагинах для обнуления массивов, сохранения статистик и других манипуляций, необходимых при выходе игрока с сервера.  
  
Инфо из **amxmodx.inc:**  
```
/* Called when client is disconnecting from server. */  
forward client_disconnect(id);
```
  
  
**Синтаксис:**  

public client_connect ( id )
  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
  
#define PLUGIN "When player disconnect"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
}  
public client_disconnect(id){  
    new name[32]  
    get_user_name(id,name,31)  
    server_print("Player %s disconnected",name)  
}
```
  
  
**Описание:**  
В этом примере при выходе игрока с сервера мы получаем его имя и выводим его в консоль сервера **server_print("Player %s disconnected",name)**  
Почему в консоль сервера, да потому что игрок-то уже отключился и эффекта никакого не будет. А вот в консоле сервера мы это увидим.
