# xs_fsign
#### Syntax
```
stock xs_fsign(Float:num)
```

#### Usage
num | ```Number to get the sign from.```
---|---
#### Description
```
Gets the sign of a float value.
```

#### Return
```
-1 if the number is negative,
0 if the number is equal to 0,
1 if the number is positive.
```


This code is a part of xs.inc. To use this code you should include xs.inc as ```#include <xs>```
