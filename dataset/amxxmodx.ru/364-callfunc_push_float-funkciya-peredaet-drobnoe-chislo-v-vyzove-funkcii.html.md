# callfunc_push_float Функция передает дробное число в вызове функции.
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Что бы передать данные с использованием функции [**callfunc_begin**](http://amxxmodx.ru/core/amxmodxinc/300-callfunc_begin-funkciya-vyzyvaet-druguyu-funkciyu-po-ee-imeni-v-etom-ili-drugom-plagine.html) необходимо использовать функции **callfunc_push_*** , каждая из них передает свой тип данных. Функция **callfunc_push_float** передает дробное число.  
  
Инфо из **amxmodx.inc:**  
```
/* Push a parameter (integer, string, float)  
 * Note that none of these values are const.  
 * Anything pushed by intrf, floatrf, array, or str  
 *  can be modified by the called function.  
 */  
native callfunc_push_float(Float: value);
```
  
  
**Синтаксис:**  

callfunc_push_float(Float: value)
  

    * **Float: value** - дробное число или переменная содержащая дробную переменную  

  
  
**Тип функции:**  
Native
