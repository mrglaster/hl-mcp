# get_concmdsnum Функция получает количество зарегистрированных консольных команд
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Применяется данная функция например в плагине adminhelp. что бы понять на какое количество страниц необходимо разбить список команд.  
  
Инфо из **amxmodx.inc:**  
```
/* Returns number of registered console commands. */  
native get_concmdsnum(flag,id = -1);
```
  
  
**Синтаксис:**  

get_concmdsnum(flag,id = -1)
  

    * **flag,id = -1** - Переменная с флагами определенного игрока или -1 (все флаги)  

Функция возвращает количество зарегистрированных консольных команд в зависимости от флагов игрока.  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
  
#define PLUGIN "[http://amxxmodx.ru] get_concmdsnum"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
      
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
    //new flags = get_user_flags(id)  
    new flags = -1  
    new cmdsnum = get_concmdsnum(flags)  
      
    server_print("^n^nComands:%d^n^n",cmdsnum)  
}  

```
  
  
**Описание:**  
Пример получит количество консольных команд и выведет их количество в консоль сервера.  
В плагине закоментирована строчка с получением флага игрока, можете создать отдельную функцию и попробовать получать количество команд для игрока, меняя ему количество флагов.
