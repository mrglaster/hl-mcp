# REU_GetAuthKey
#### Syntax
```
native REU_GetAuthKey(const index, dest[], maxlen);
```

#### Usage
index | ```Client index```
---|---
index | ```Buffer to copy the authkey```
index | ```Maximum buffer size```
#### Description
```
Get client authkey
```

#### Return
```
Number of cells copied to buffer
```
