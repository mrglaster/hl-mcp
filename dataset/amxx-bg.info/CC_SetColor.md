# CC_SetColor
#### Syntax
```
stock CC_SetColor(index, bool:force = false)
```

#### Usage
index | ```CC_COLOR_* or player index to get color from```
---|---
force | ```If set to true, custom colors in the code will be ignored and the message will be forced to use the color set here```
#### Description
```
Sets the team color for the next message that's going to be sent.
```

#### Return
```
This function has no return value.
```


This code is a part of cromchat.inc. To use this code you should include cromchat.inc as ```#include <cromchat>```
