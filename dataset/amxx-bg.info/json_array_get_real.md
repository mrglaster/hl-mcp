# json_array_get_real
#### Syntax
```
native Float:json_array_get_real(const JSON:array, index);
```

#### Usage
array | ```Array handle```
---|---
index | ```Position in the array (starting from 0)```
#### Description
```
Gets a real number from the array.
```

#### Return
```
The number as float
```

#### Error
```
If passed handle is not a valid array
```


This code is a part of json.inc. To use this code you should include json.inc as ```#include <json>```
