# json_free
#### Syntax
```
native bool:json_free(&JSON:handle);
```

#### Usage
handle | ```JSON handle to be freed```
---|---
#### Description
```
Frees handle.
```

#### Return
```
True if succeed, false otherwise
```

#### Error
```
If passed handle is not a valid handle
```


This code is a part of json.inc. To use this code you should include json.inc as ```#include <json>```
