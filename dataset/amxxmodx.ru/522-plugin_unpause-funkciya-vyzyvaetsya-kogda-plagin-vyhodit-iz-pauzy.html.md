# plugin_unpause Функция вызывается когда плагин выходит из паузы
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Форвард plugin_unpause необходим для отлова момента когда плагин выходит из состояния паузы, в которое он может попасть например с помощью функции [**plugin_pause**](http://amxxmodx.ru/core/amxmodxinc/177-plugin_pause-funkciya-vyzyvaetsya-kogda-plagin-perehodit-v-sostoyanie-pauzy.html)  
  
Инфо из **amxmodx.inc:**  
```
/* Called when the plugin is unpaused. */  
forward plugin_unpause();
```
  
  
**Синтаксис:**  

plugin_unpause()
  
  
**Тип функции:**  
Forward
