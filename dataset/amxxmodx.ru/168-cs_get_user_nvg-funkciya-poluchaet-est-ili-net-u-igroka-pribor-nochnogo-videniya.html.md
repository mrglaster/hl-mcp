# cs_get_user_nvg Функция получает есть или нет у игрока прибор ночного видения
[CStrike](http://amxxmodx.ru/cstrike/)
Когда я играю, я никогда не пользуюсь ночным видением, кроме игры на серверах зомби, когда там кромешная темнота, ночное видение там в специальных предметах лежит и очень ценное, и очень неприятно когда покупаешь то что у тебя уже есть. С помощью функции можно проверить есть ли у игрока прибор ночного видения и не позволить ему купить его еще раз.  
  
Инфо из **cstrike.inc:**  
```
/* Does user have night vision goggles?  
 */  
native cs_get_user_nvg(index);
```
  
  
**Синтаксис:**  

cs_get_user_nvg ( index )
  

    * **index** id игрока  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <cstrike>  
  
#define PLUGIN "[cstrike.inc] cs_get_user_nvg"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
  
    register_clcmd("say /get_nvg","get_nvg")  
      
}  
public get_nvg(id){  
    //Регистрациая новой переменной  
    new nvg  
    //Запись в переменную информации о ночном видение для игрока  
    nvg = cs_get_user_nvg(id)  
    //Вывод сообщения в чат.  
    client_print(id,print_chat,"[cstrike.inc] You %s night vision",nvg ? "have":"dont`t have")  
  
}
```
  
  
**Описание:**  
Очередная не сложная функция получающая информацию о игроке.  
Все как всегда, регистрируем переменную и получаем 1 или 0 ( есть или нет) ночное видение у игрока.  
Для визуализации с помощью функции **client_print** , организован вывод текстового сообщения в чат.
