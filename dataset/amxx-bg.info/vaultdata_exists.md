# vaultdata_exists
#### Syntax
```
native vaultdata_exists(const key[]);
```

#### Usage
key | ```Key to check```
---|---
#### Description
```
Checks if a key exists in the vault.
```

#### Return
```
1 if an entry was found, 0 otherwise.
```


This code is a part of vault.inc. To use this code you should include vault.inc as ```#include <vault>```
