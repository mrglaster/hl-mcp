# abs
#### Syntax
```
stock abs(x)
```

#### Usage
x | ```Integral value```
---|---
#### Description
```
Returns the absolute value of a number.
```

#### Return
```
Absolute value of x (x if it is greater than 0, -x otherwise)
```


This code is a part of core.inc. To use this code you should include core.inc as ```#include <core>```
