et_user_noclip
[Fun](http://127.0.0.1:8000/content/index.htm) (fun.inc) 
Description
set_user_noclip - Sets player noclip. If you want to disable noclip set only first parameter. 
Syntax
set_user_noclip ( index, [ noclip = 0 ] ) 
Type
Native 
Notes
index is a player index from 1 to 32.   
  
Setting the second parameter sets noclip on.
