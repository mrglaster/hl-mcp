# ns_set_score
#### Syntax
```
native ns_set_score(idPlayer, score);
```

#### Usage
idPlayer | ```The player to get the score for.```
---|---
score | ```What to set the player's score as.```
#### Description
```
Sets the player's score.
```

#### Note
```
The score from level is automatically factored into the scoreboard in combat.
```

#### Return
```
This function has no return value.
```


This code is a part of ns.inc. To use this code you should include ns.inc as ```#include <ns>```


  
  

Warning: This include is compatible only with Natural Selection and will not function with other mods (e.g., Valve, Counter Strike 1.6, etc.).
