# fflush
#### Syntax
```
native fflush(file);
```

#### Usage
file | ```File handle, or 0 for all open streams```
---|---
#### Description
```
Flushes a buffered output stream.
```

#### Return
```
0 on success, -1 on failure
```


This code is a part of file.inc. To use this code you should include file.inc as ```#include <file>```
