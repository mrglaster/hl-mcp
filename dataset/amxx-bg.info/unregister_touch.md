# unregister_touch
#### Syntax
```
native unregister_touch(registerid);
```

#### Usage
registerid | ```Touch forward id```
---|---
#### Description
```
Removes a previously registered touch hook.
```

#### Return
```
1 on success, 0 if nothing was removed
```


This code is a part of engine.inc. To use this code you should include engine.inc as ```#include <engine>```
