# strcat
#### Syntax
```
native strcat(dest[], const source[], maxlength);
```

#### Usage
dest | ```String to append to.```
---|---
source | ```Source string to concatenate.```
maxlength | ```Maximum length of entire buffer.```
#### Description
```
Concatenates one string onto another.
```

#### Return
```
Number of bytes written.
```


This code is a part of string.inc. To use this code you should include string.inc as ```#include <string>```
