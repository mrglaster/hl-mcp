# json_get_parent
#### Syntax
```
native JSON:json_get_parent(const JSON:value);
```

#### Usage
value | ```JSON handle```
---|---
#### Description
```
Gets value's parent handle.
```

#### Note
```
Parent's handle needs to be freed using json_free() native.
```

#### Return
```
Parent's handle
```


This code is a part of json.inc. To use this code you should include json.inc as ```#include <json>```
