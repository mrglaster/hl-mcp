# rg_find_weapon_bpack_by_name
#### Syntax
```
native rg_find_weapon_bpack_by_name(const index, const weapon[]);
```

#### Usage
index | ```Client index```
---|---
weapon | ```Weapon name```
#### Description
```
Finds the weapon by name in the player's inventory.
```

#### Return
```
Weapon's entity index, 0 otherwise
```


This code is a part of reapi_gamedll.inc. To use this code you should include reapi_gamedll.inc as ```#include <reapi_gamedll>```
