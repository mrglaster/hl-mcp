# cshop_has_item
#### Syntax
```
native bool:cshop_has_item(id, item)
```

#### Usage
id | ```Player's index.```
---|---
item | ```Item ID.```
#### Description
```
Checks whether a player has an item or not.
```

#### Return
```
True if he has, false if he hasn't.
```


This code is a part of customshop.inc. To use this code you should include customshop.inc as ```#include <customshop>```
