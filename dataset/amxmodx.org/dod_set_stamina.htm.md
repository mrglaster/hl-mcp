od_set_stamina
[DoDFun](http://127.0.0.1:8000/content/index.htm) (dodfun.inc) 
Description
dod_set_stamina - Sets a player's stamina 
Syntax
dod_set_stamina ( player, set = STAMINA_SET, minvalue = 0, maxvalue = 100 ) 
Type
Native 
Notes
Example: for full stamina use dod_set_stamina(1,STAMINA_SET,100,100)   
  
Value is from 0 - 100
