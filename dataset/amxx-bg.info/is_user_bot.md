# is_user_bot
#### Syntax
```
native is_user_bot(index);
```

#### Usage
index | ```Client index```
---|---
#### Description
```
Returns if the client is a bot.
```

#### Return
```
1 if client is a bot, 0 otherwise
```


This code is a part of amxmodx.inc. To use this code you should include amxmodx.inc as ```#include <amxmodx>```
