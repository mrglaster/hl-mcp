s_set_user_plant
[Cstrike](http://127.0.0.1:8000/content/index.htm) (cstrike.inc) 
Description
cs_set_user_plant - Sets whether a player can plant the bomb. 
Syntax
cs_set_user_plant ( index, [ plant = 1, showbombicon = 1 ] ) 
Type
Native 
Notes
This will only work if the player is at a bomb site and has the bomb. You should use this if you give a player weapon_c4 manually.   
  
If showbombicon is 1, the green C4 icon will be shown on the user's HUD.
