# json_get_type
#### Syntax
```
native JSONType:json_get_type(const JSON:value);
```

#### Usage
value | ```JSON handle```
---|---
#### Description
```
Gets JSON type of passed value.
```

#### Return
```
JSON type (JSONType constants)
```

#### Error
```
If a value handle is invalid
```


This code is a part of json.inc. To use this code you should include json.inc as ```#include <json>```
