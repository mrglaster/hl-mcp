[TSX](http://127.0.0.1:8000/content/index.htm) (tsx.inc) 
Description
ts_getuserwpn - Gets info about the user's current weapon. 
Syntax
ts_getuserwpn ( index, &clip, &ammo, &mode, &extra ) 
Type
Native 
Notes
Returns the index of the current weapon. Refer to the TSW_* constants in tsconst.inc for the return value. mode is the mode of fire and extra are the extras such as silencer and flashlight.
