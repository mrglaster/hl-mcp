# cshop_get_float
#### Syntax
```
native Float:cshop_get_float(item, setting[])
```

#### Usage
item | ```Item id.```
---|---
setting | ```Setting's name.```
#### Description
```
Returns a float value from a specific item's setting.
```

#### Return
```
Setting's value.
```


This code is a part of customshop.inc. To use this code you should include customshop.inc as ```#include <customshop>```
