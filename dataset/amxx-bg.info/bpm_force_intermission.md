# bpm_force_intermission
#### Syntax
```
native bpm_force_intermission()
```

#### Description
```
Forces the plugin to run the intermission event.
```

#### Return
```
This function has no return value.
```


This code is a part of bestplayer.inc. To use this code you should include bestplayer.inc as ```#include <bestplayer>```


  
  

Warning! This is an external include! It does not come bundled with AMX Mod X "out of the box" and is provided with basebuilder mod.
