# rg_has_item_by_name
#### Syntax
```
native bool:rg_has_item_by_name(const index, const item[]);
```

#### Usage
index | ```Client index```
---|---
item | ```Item name```
#### Description
```
Checks if the player has the item.
```

#### Return
```
true if he does, false otherwise
```
