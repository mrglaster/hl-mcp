# AddCommas
#### Syntax
```
stock AddCommas(iNum, szOutput[], iLen)
```

#### Usage
iNum | ```Number.```
---|---
szOutput | ```The buffer to store string in.```
iLen | ```Character size of the output buffer.```
#### Description
```
Add commas in number and convert that to string.
```

#### Return
```
Number of cells written.
```
