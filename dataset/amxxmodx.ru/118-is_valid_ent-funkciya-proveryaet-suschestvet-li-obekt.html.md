# is_valid_ent Функция проверяет существует ли объект
[The Engine Module](http://amxxmodx.ru/engine/) [engine.inc](http://amxxmodx.ru/engine/engineinc/)
Данная функция предназначена для получения ответа на вопрос:  
Существует ли объект с конкретным индексом или нет.  
  
Инфо из **engine.inc:**  
```
/* Is entity valid? */  
native is_valid_ent(iIndex);
```
  
  
**Синтаксис:**  

is_valid_ent ( entity )
  
Возвращает 1 если объект существует, 0 если нет.  
  
**Тип функции:**  
Native  
  
**Пример и описание:**  
Смотри в описание функции [create_entity](http://amxxmodx.ru/engine/117-create_entity-funkciya-sozdaet-obekt-i-vozvraschaet-ego-indeks.html).
