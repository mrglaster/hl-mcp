# json_array_append_real
#### Syntax
```
native bool:json_array_append_real(JSON:array, Float:number);
```

#### Usage
array | ```Array handle```
---|---
number | ```Real number to set```
#### Description
```
Appends a real number in the array.
```

#### Return
```
True if succeed, false otherwise
```

#### Error
```
If passed handle is not a valid array
```


This code is a part of json.inc. To use this code you should include json.inc as ```#include <json>```
