# set_user_godmode
#### Syntax
```
native set_user_godmode(index, godmode = 0);
```

#### Usage
index | ```Client index```
---|---
godmode | ```1 to enable godmode, 0 to disable```
#### Description
```
Sets player's godmode.
```

#### Return
```
This function has no return value.
```

#### Error
```
If player is not connected or not within the range
of 1 to MaxClients.
```


This code is a part of fun.inc. To use this code you should include fun.inc as ```#include <fun>```
