# xs_deg2rad
#### Syntax
```
stock Float:xs_deg2rad(Float:x)
```

#### Usage
x | ```Input degrees.```
---|---
#### Description
```
Converts degrees to radians.
```

#### Return
```
Degrees converted to radians.
```


This code is a part of xs.inc. To use this code you should include xs.inc as ```#include <xs>```
