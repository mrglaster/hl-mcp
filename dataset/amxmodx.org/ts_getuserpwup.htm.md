s_getuserpwup
[TSX](http://127.0.0.1:8000/content/index.htm) (tsx.inc) 
Description
ts_getuserpwup - Returns a user's powerup and amount. 
Syntax
ts_getuserpwup ( index, &Value ) 
Type
Native 
Notes
Refer to the TSPWUP_* constants in tsconst.inc for the return value. The amount of powerup remaining (ie: time left, in seconds) is passed into Value.
