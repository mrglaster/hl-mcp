# was_user_zombie
#### Syntax
```
native was_user_zombie(id)
```

#### Usage
id | ```Индекс на играча```
---|---
#### Description
```
Връща съответна стойност, ако играчът е бил зомби при раждането си
```

#### Return
```
Връща true ако е бил зомби и false - ако не е бил. (bool)
```
