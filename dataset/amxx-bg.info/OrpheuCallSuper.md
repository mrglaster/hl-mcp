# OrpheuCallSuper
#### Syntax
```
native OrpheuCallSuper(OrpheuFunction:function,any:...)
```

#### Usage
function | ```A handler to the function```
---|---
any | ```The arguments of the function```
#### Description
```
Calls a function and triggers its hooks
```


This code is a part of orpheu.inc. To use this code you should include orpheu.inc as ```#include <orpheu>```
