# is_deal_offer
#### Syntax
```
native is_deal_offer(itemid)
```

#### Description
```
Връща съответна стойност дали итемът е в офертите на "Round Deal" в съответния рунд
```

#### Return
```
Връща true ако е и false - ако не е. (bool)
```
