# ts_weaponspawn
#### Syntax
```
stock ts_weaponspawn(const weaponid[], const duration[], const extraclip[], const spawnflags[], const Float:Origin[3])
```

#### Description
```
Spawns a Weapon
```


This code is a part of tsx.inc. To use this code you should include tsx.inc as ```#include <tsx>```. The include is compatible only with The Specialists mod and won't work with other games or mods (Counter Strike 1.6, Half-Life, DoD, etc)
