# cs_set_user_submodel Функция Устанавливает подмодель игроку.
[CStrike](http://amxxmodx.ru/cstrike/)
Подмодель - щипчики у контров и бомба у террористов. При этом это только модель, самого предмета у игрока нет.  
  
Инфо из **cstrike.inc:**  
```
/* Sets the submodel setting of the player.  
 * If this is 1, then the user has a backpack or defuser on their model (depending on team)  
 * 0 removes it.  
 */  
native cs_set_user_submodel(index, value);
```
  
  
**Синтаксис:**  

cs_set_user_submodel(index, value)
  

    * **index** - Индекс игрока  

    * **value** - Значение для модели( 1 установить / 0 убрать)  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <cstrike>  
  
#define PLUGIN "[cstrike.inc] cs_set_user_submodel"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
  
    register_clcmd("say /asm","AddSubModel")  
}  
public AddSubModel(){  
  
    new Players[32]   
    new playerCount, i, id   
    get_players(Players, playerCount, "ah")   
      
    for (i=0; i<playerCount; i++) {  
        id = Players[i]  
  
        cs_set_user_submodel(id,1)  
          
    }  
      
}
```
  
  
**Описание:**  
По умолчанию у игроков нет саб моделей, кроме как одного террориста на de_* картах.  
По этому в примере по команде **say /asm** всем присутствующим игрокам на сервере выдается саб модель.  
[**Цикл for**](http://amxxmodx.ru/lessons-on-pawn/211-urok-5-cikly-kakie-byvayut-cikly-chto-takoe-forwhiledo-while.html) необходим для выдачи всем игрокам.
