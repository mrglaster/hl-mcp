# UpdateFrags
#### Syntax
```
stock UpdateFrags(iAttacker, iVictim, iFrags, iDeaths, iScoreboard)
```

#### Usage
iAttacker | ```Attacker index, 0 = Skip.```
---|---
iVictim | ```Victim index, 0 = Skip.```
iFrags | ```Frags.```
iDeaths | ```Deaths.```
iScoreboard | ```Update frags and deaths on scoreboard (1 | 0).```
#### Description
```
Update Player Frags and Deaths.
```

#### Return
```
This function has no return value.
```
