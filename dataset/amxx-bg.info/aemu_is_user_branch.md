# aemu_is_user_branch
#### Syntax
```
native bool:aemu_is_user_branch(const index, BranchType:type);
```

#### Usage
index | ```Client index```
---|---
type | ```Use the BRANCH_* BranchType enum```
#### Description
```
Determine whether specified branch is in use.
```

#### Return
```
return true if so, false otherwise
```


This code is a part of authemu.inc. To use this code you should include authemu.inc as ```#include <authemu>```
