# CreateLangKey
#### Syntax
```
native TransKey:CreateLangKey(const key[]);
```

#### Usage
key | ```Key to create or find```
---|---
#### Description
```
Creates a new or finds an existing translation key.
```

#### Return
```
Key index
```


This code is a part of lang.inc. To use this code you should include lang.inc as ```#include <lang>```
