# json_object_clear
#### Syntax
```
native bool:json_object_clear(JSON:object);
```

#### Usage
object | ```Object handle```
---|---
#### Description
```
Removes all keys and their values in the object.
```

#### Return
```
True if succeed, false otherwise
```

#### Error
```
If passed handle is not a valid object
```


This code is a part of json.inc. To use this code you should include json.inc as ```#include <json>```
