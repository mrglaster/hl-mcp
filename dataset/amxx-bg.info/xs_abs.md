# xs_abs
#### Syntax
```
stock xs_abs(num)
```

#### Usage
num | ```Number to get the absolute value from.```
---|---
#### Description
```
Gets the absolute value of a number.
```

#### Return
```
Absolute value of the input number.
```


This code is a part of xs.inc. To use this code you should include xs.inc as ```#include <xs>```
