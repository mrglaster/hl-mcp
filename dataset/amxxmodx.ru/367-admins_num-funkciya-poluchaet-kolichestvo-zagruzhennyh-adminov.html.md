# admins_num Функция получает количество загруженных админов
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Работать с функцией **admins_num** приходилось не многим, так как в подавляющем большинстве администраторы игровых серверов предпочитают пользоваться готовыми решения для управления игровыми админами. К стати описания этой функции отсутствует даже на официальном сайте.  
  
Инфо из **amxmodx.inc:**  
```
/**  
 * Gets the number of admins in the dynamic admin  
 * storage list  
 */  
native admins_num();
```
  
  
**Синтаксис:**  

admins_num()
  
Функция вернет количество загруженных администраторов.  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
  
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
  
#define PLUGIN "[amxmodx.inc] admins_num"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
    register_srvcmd("count_admins","c_admins")  
}  
public c_admins(){  
  
    //Регистрация новой переменной и   
    //получение количества загруженных админов  
    new admin_num = admins_num()  
    //Вывод результата в консоль сервера  
    server_print("^n^nadmins_num[%d]^n^n",admin_num)  
      
}
```
