# xs_seed
#### Syntax
```
stock xs_seed(seed)
```

#### Usage
x | ```The seed to set.```
---|---
#### Description
```
Sets the seed for the random number generation.
```

#### Return
```
This function has no return value.
```


This code is a part of xs.inc. To use this code you should include xs.inc as ```#include <xs>```
