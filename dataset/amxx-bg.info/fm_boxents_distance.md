# fm_boxents_distance
#### Syntax
```
stock Float:fm_boxents_distance(boxent1, boxent2) {
```

#### Description

Allows to check if some entity is in another entity. Example: player's coordinates is within the box

```
if( ! fm_boxents_distance(player, box) )
{
}
```


This code is a part of fakemeta_util.inc. To use this code you should include fakemeta_util.inc as ```#include <fakemeta_util>```
