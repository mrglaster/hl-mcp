# set_user_noclip
#### Syntax
```
native set_user_noclip(index, noclip = 0);
```

#### Usage
index | ```Client index```
---|---
noclip | ```1 to enable noclip, 0 to disable```
#### Description
```
Enables or disables player's noclip.
```

#### Return
```
This function has no return value.
```

#### Error
```
If player is not connected or not within the range
of 1 to MaxClients.
```


This code is a part of fun.inc. To use this code you should include fun.inc as ```#include <fun>```
