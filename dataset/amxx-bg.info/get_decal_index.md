# get_decal_index
#### Syntax
```
native get_decal_index(const szDecalName[]);
```

#### Usage
szDecalName | ```Decal name```
---|---
#### Description
```
Returns the index of a decal.
```

#### Return
```
Decal index >= 0, or -1 if decal was not found
```


This code is a part of engine.inc. To use this code you should include engine.inc as ```#include <engine>```
