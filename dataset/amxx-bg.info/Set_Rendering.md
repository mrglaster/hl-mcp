# Set_Rendering
#### Syntax
```
stock Set_Rendering(ent, fx = kRenderFxNone, r = 255, g = 255, b = 255, render = kRenderNormal, amount = 16)
```

#### Usage
ent | ```Entity index.```
---|---
fx | ```Render fx, check kRendeFx in amxconst.inc```
r | ```Red color (0..255)```
g | ```Green color (0..255)```
b | ```Blue color (0..255)```
render | ```Render mode, check kRender in amxconst.inc```
amout | ```Render amout (0..255)```
#### Description
```
Set rendering of entity.
```

#### Return
```
This function has no return value.
```
