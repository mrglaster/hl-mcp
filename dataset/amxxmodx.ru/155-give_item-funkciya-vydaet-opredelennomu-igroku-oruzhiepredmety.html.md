# give_item Функция выдает определенному игроку оружие/предметы
[The Fun Module](http://amxxmodx.ru/fun/) [fun.inc](http://amxxmodx.ru/fun/funinc/)
В различных VIP плагинах игроку с данным статусом присваиваются различные привилегии, в том числе выдача различных видов оружия при появлении.  
Данная функция как раз поможет вам выдать необходимый предмет в нужный момент.  
  
Инфо из **fun.inc:**  
```
/* Gives item to player, name of item can start  
 * with weapon_, ammo_ and item_. This event  
 * is announced with proper message to all players. */  
native give_item(index, const item[]);
```
  
  
**Синтаксис:**  

give_item ( index, const item[] )
  

    * **index** - id игрока кому выдать предмет  

    * **const item[]** - константа предмета ( weapon_* оружие, ammo_* патроны и item_* предметы)  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <fun>  
  
#define PLUGIN "[fun.inc] give_item"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
    register_clcmd("say /give_awp","give_awp")  
}  
public give_awp(id){  
    give_item(id,"weapon_awp")  
}  

```
  
  
**Описание:**  
Появились на респе, первый раунд? - Пишите скорее /give_awp и пока все с пистолетами, у вас уже есть "Слон".  
Такое поведение не одобряется большинством адекватных игроков, по этому воспринимайте это неудачной шуткой и рабочим примером использования функции.
