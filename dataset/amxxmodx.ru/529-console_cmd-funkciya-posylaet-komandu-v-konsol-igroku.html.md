# console_cmd Функция посылает команду в консоль игроку
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Иногда требуется сделать что то за игрока, от его лица, например заставить прыгнуть. С помощью функции **console_cmd** это сделать будет очень просто.  
  
Инфо из **amxmodx.inc:**  
```
/* Sends command to console. */  
native console_cmd(id,const cmd[],any:...);
```
  
  
**Синтаксис:**  

console_cmd(id,const cmd[],any:...);
  

    * **id** - Индекс игрока  

    * **const cmd[]** - Команда  

    * **any:...** - данные для подстановки  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <sockets>  
  
#define PLUGIN "[http://amxxmodx.ru] console_cmd"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
    register_clcmd("say jump","jump")  
      
}  
  
public jump(id){  
      
    console_cmd(id,"+jump;")  
      
}
```
  
  
**Описание:**  
Написав в чат jump вы один раз подпрыгните, но обратите внимание, пока вы не сделаете команду -jump вы не сможете прыгать.
