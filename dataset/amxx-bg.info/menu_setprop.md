# menu_setprop
#### Syntax
```
native menu_setprop(menu, prop, ...);
```

#### Usage
menu | ```Menu resource identifier.```
---|---
prop | ```MPROP_ constant.```
... | ```Property parameters.```
#### Description
```
Sets a menu property.
```

#### Return
```
1 on success, 0 on failure.
```

#### Error
```
Invalid menu resource or property.
```


This code is a part of newmenus.inc. To use this code you should include newmenus.inc as ```#include <newmenus>```
