# menu_item_setname
#### Syntax
```
native menu_item_setname(menu, item, const name[]);
```

#### Usage
menu | ```Menu resource identifier.```
---|---
item | ```Item identifier.```
name | ```New item display text.```
#### Description
```
Sets an item's display text.
```

#### Return
```
1 on success, 0 on failure.
```

#### Error
```
Invalid menu resource.
```


This code is a part of newmenus.inc. To use this code you should include newmenus.inc as ```#include <newmenus>```
