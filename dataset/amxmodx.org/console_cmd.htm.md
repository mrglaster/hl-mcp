[Core](http://127.0.0.1:8000/content/index.htm) (amxmodx.inc) 
Description
console_cmd - Sends a command to a player's console 
Syntax
console_cmd ( id, const cmd[], ... ) 
Type
Native 
Notes
id is a player index from 1 to 32. 0 will send to all players.   
  
cmd can be a formatted string in the style of [format](http://127.0.0.1:8000/content/format.htm).
