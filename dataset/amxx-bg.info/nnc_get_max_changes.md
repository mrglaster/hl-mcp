# nnc_get_max_changes
#### Syntax
```
native nnc_get_max_changes()
```

#### Description
```
Returns how many times players can change their name.
```

#### Return
```
Number of maximum allowed name changes
```


This code is a part of nnc.inc. To use this code you should include nnc.inc as ```#include <nnc>```


  
  

Warning! This is an external include! It does not come bundled with AMX Mod X "out of the box" and requires additional installation. Use it only if you are absolutely certain that you need it.
