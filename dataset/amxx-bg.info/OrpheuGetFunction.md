# OrpheuGetFunction
#### Syntax
```
native OrpheuFunction:OrpheuGetFunction(const libFunctionName[],const className[]="")
```

#### Usage
libFunctionName | ```The name of the function as it is in the file where the function is defined```
---|---
className | ```The name of the class if the function belongs to one```
#### Description
```
Retrieves a function based on a function name
The name must be the same as the one in the file where the function is defined
```

#### Return
```
A handler to the function
```


This code is a part of orpheu.inc. To use this code you should include orpheu.inc as ```#include <orpheu>```
