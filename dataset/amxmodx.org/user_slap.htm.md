[Core](http://127.0.0.1:8000/content/index.htm) (amxmodx.inc) 
Description
user_slap - Slaps a player for variable damage (power). 
Syntax
user_slap ( index, power, [ rnddir ] ) 
Type
Native 
Notes
If rnddir is specified as 0, the player is not thrown in a random direction.   
  
index,is a player index from 1 to 32.
