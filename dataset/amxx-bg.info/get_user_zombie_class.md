# get_user_zombie_class
#### Syntax
```
native get_user_zombie_class(id)
```

#### Usage
id | ```Индекс на играча```
---|---
#### Description
```
Връща индекса на зомби класа на играча. (int)
```

#### Return
```
Връща индекса на зомби класа на играча (-1 ако е Classic Zombie)
```
