# set_user_fire_duration
#### Syntax
```
native set_user_fire_duration(id, duration)
```

#### Usage
id | ```Индекс на играча```
---|---
duration | ```Стойността на oставащата кръв за взимане при горене, която искаме да зададем (1 - за да спрем горенето на играч).```
#### Description
```
Задава стойността на oставащата кръв за взимане при горене
```


This code is a part of vzbb_mod.inc. To use this code you should include vzbb_mod.inc as ```#include <vzbb_mod>```
