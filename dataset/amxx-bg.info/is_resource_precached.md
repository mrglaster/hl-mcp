# is_resource_precached
#### Syntax
```
native bool:is_resource_precached(szItem[], iType)
```

#### Usage
szItem[] | ```Buffer to copy resource name.```
---|---
iType | ```Type of the resource. See _:Types Enum```
#### Description
```
Returns if the resource is precached.
```

#### Type
```
Boolean
```

#### Return
```
True if it is, false otherwise.
```


This code is a part of precache_list.inc. To use this code you should include precache_list.inc as ```#include <precache_list>```


  
  

Warning! This is an external include! It does not come bundled with AMX Mod X "out of the box" and requires additional installation. Use it only if you are absolutely certain that you need it.
