et_user_menu
[Core](http://127.0.0.1:8000/content/index.htm) (amxmodx.inc) 
Description
get_user_menu - Gets what menu the player is watching and what keys for menu he have. 
Syntax
get_user_menu ( index,&id,&keys ) 
Type
Native 
Notes
When there is no menu the index is 0. If the id is negative then the menu is VGUI in other case the id is from [register_menuid](http://127.0.0.1:8000/content/register_menuid.htm)() function.
