# Функция преобразует число в слово.
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Функция преобразует число в слово из диапазона от 0 до 999.  
  
Инфо из **amxmodx.inc:**  
```
/* Converts numbers from range 0 - 999 to words. */  
native num_to_word(num,output[],len);
```
  
  
**Синтаксис:**  

num_to_word(num,output[],len)
  

    * **num** - Число для преобразования  

    * **output[]** - Массив с преобразованным в текст числом  

    * **len** - Максимальный размер массива  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
  
#define PLUGIN "[http://amxxmodx.ru] num_to_word"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
    new data[256]  
    new num = 115  
      
    num_to_word(num,data,charsmax(data))  
      
    server_print("[%d] -- [%s]",num,data)  
}
```
  
  
**Описание:**  
В примере преобразуется число 115 в текст, в консоли сервера вы получите такой результат:  

[115] -- [one hundred fifteen ]
  
То есть число 115 написанное словами.
