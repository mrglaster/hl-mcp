# get_user_wlstats
#### Syntax
```
native get_user_wlstats(index,wpnindex,stats[DODX_MAX_STATS],bodyhits[MAX_BODYHITS]);
```

#### Description
```
Gets life (from spawn to spawn) stats from given weapon index.
```


This code is a part of dodstats.inc. To use this code you should include dodstats.inc as ```#include <dodstats>```


  
  

Warning: This include is compatible only with Day of Defeat and will not function with other mods (e.g., Valve, Counter Strike 1.6, etc.).
