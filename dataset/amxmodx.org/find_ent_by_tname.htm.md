find_ent_by_tname - Finds an entity in the world by target name, returning the index. 
Syntax
find_ent_by_tname ( StartEntity, Classname[] ) 
Type
Native 
Notes
If none is found, 0 is returned. If starting entity should not be specified, use -1.
