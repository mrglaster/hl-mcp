# get_user_noclip
#### Syntax
```
native get_user_noclip(index);
```

#### Usage
index | ```Client index```
---|---
#### Description
```
Gets whether a player has noclip enabled or not.
```

#### Return
```
1 if noclip is enabled, 0 if disabled
```

#### Error
```
If player is not connected or not within the range
of 1 to MaxClients.
```


This code is a part of fun.inc. To use this code you should include fun.inc as ```#include <fun>```
