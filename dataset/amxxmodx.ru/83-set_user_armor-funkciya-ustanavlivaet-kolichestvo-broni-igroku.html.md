# set_user_armor Функция устанавливает количество брони игроку
[The Fun Module](http://amxxmodx.ru/fun/) [fun.inc](http://amxxmodx.ru/fun/funinc/)
В плагинах для выдачи брони, обычно используют функцию **cs_set_user_armor** , так как в ней можно указать не только обычную броню, но и шлем.  
По этому я буквально 2 слова скажу о функции **set_user_armor**.  
  
Инфо из **fun.inc:**  
```
/* Sets player armor. */  
native set_user_armor(index, armor);
```
  
  
**Синтаксис:**  

set_user_armor ( index, armor )
  

    * **index** - id игрока кому устанавливаем оброню  

    * **armor** - количество брони  

  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <fun>  
  
#define PLUGIN "Set user armor"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
      
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
    register_clcmd("say /give-armor","set_armor")  
}  
public set_armor(id){  
          
    set_user_armor(id,102)  
}  

```
  
**Описание:**  
В рабочей функции нашей команды у нас есть только лишь рассматриваемая функция, которая и устанавливает нам броню в размере 102.  
Напомню, что лучше пользоваться функцией **cs_set_user_armor**.
