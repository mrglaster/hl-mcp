og_to_file
[Core](http://127.0.0.1:8000/content/index.htm) (amxmodx.inc) 
Description
log_to_file - Logs something to a specified file. 
Syntax
log_to_file ( const file[], const message[], [ ... ] ) 
Type
Native 
Notes
Message can be formatted according to [format](http://127.0.0.1:8000/content/format.htm).   
  
Note: if no path is specified, the file is put into the amxmodx logs folder.
