# xs_sqrt
#### Syntax
```
stock Float:xs_sqrt(Float:x)
```

#### Usage
x | ```The input value.```
---|---
#### Description
```
Calculates the square root of the input value.
```

#### Note
```
This is an alias for floatsqroot().
```

#### Return
```
The square root of the input value.
```


This code is a part of xs.inc. To use this code you should include xs.inc as ```#include <xs>```
