# cshop_give_points
#### Syntax
```
native cshop_give_points(id, points)
```

#### Usage
id | ```Player's index.```
---|---
points | ```Amount of points.```
#### Description
```
Give/take points to/from a player.
```

#### Return
```
This function has no return value.
```


This code is a part of customshop.inc. To use this code you should include customshop.inc as ```#include <customshop>```
