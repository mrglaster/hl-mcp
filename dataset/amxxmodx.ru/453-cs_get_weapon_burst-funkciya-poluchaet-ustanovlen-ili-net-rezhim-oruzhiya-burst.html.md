# cs_get_weapon_burst Функция получает установлен или нет режим оружия burst
[CStrike](http://amxxmodx.ru/cstrike/)
Инфо из **cstrike.inc:**  
```
/* Returns 1 if specified weapon is in burst mode.  
 */  
native cs_get_weapon_burst(index);
```
  
  
**Синтаксис:**  
[code]cs_get_weapon_burst(index)/code]  

    * **index** - Индекс _оружия_ ( объекта, а не игрока его держащего)  

Возвращает 1 если есть и 0 если нет  
  
**Тип функции:**  
Native
