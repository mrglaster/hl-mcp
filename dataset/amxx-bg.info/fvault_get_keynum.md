# fvault_get_keynum
#### Syntax
```
stock fvault_get_keynum(const vaultname[], const key[])
```

#### Usage
vaultname | ```Vault name to look in```
---|---
key | ```Key name to search for```
#### Description
```
Retrieves a key number specified by its name
```

#### Return
```
Returns key number on success, -1 on failure
```


This code is a part of fvault.inc. To use this code you should include fvault.inc as ```#include <fvault>```


  
  

Warning! This is an external include! It does not come bundled with AMX Mod X "out of the box" and requires additional installation. Use it only if you are absolutely certain that you need it.
