# OrpheuSetParam
#### Syntax
```
native OrpheuSetParam(num,any:...)
```

#### Usage
num | ```The number of the argument. The first argument would be the number "1"```
---|---
any | ```Depending on the type of the argument, a value to be replace it to change the behaviour of the hooked function```
#### Description
```
Sets the value of an argument
```


This code is a part of orpheu.inc. To use this code you should include orpheu.inc as ```#include <orpheu>```
