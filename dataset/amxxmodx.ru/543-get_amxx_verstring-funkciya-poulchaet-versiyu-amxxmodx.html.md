# get_amxx_verstring Функция поулчает версию amxxmodx
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Не думаю что кому то будет полезна данная функция, но она есть и ее работа заключается лишь в том, что бы получить текущую версию amxmodx сервера.  
  
Инфо из **amxmodx.inc:**  
```
/* Returns AMXX's version string of the current gameserver */  
native get_amxx_verstring(buffer[], length);
```
  
  
**Синтаксис:**  

get_amxx_verstring(buffer[], length)
  

    * **buffer[]** - Массив для данных  

    * **length** - Его длина  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
  
#define PLUGIN "[http://amxxmodx.ru] get_amxx_verstring"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
    new data [256]  
    get_amxx_verstring(data,charsmax(data))  
      
    server_print("Amxx version: %s",data)  
      
}
```
  
  
**Описание:**  
Пример получает версию amxmodx сервера и выводит ее в консоль.
