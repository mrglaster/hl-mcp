# get_localinfo Функция получает значение localinfo.
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Функция предназначена для получения localinfo, обратите внимание что вы должны знать имя переменной.  
  
Инфо из **amxmodx.inc:**  
```
/* Gets info from server. */  
native get_localinfo(const info[],output[],len);
```
  
  
**Синтаксис:**  

get_localinfo(const info[],output[],len)
  

    * **const info[]** - Имя переменной, значение которой хотим получить.  

    * **output[]** - Массив для данных  

    * **len** - Его длина  

  
  
**Тип функции:**  
Native
