# xs_rad2deg
#### Syntax
```
stock Float:xs_rad2deg(Float:x)
```

#### Usage
x | ```Input radians.```
---|---
#### Description
```
Converts radians to degrees.
```

#### Return
```
Radians converted to degrees.
```


This code is a part of xs.inc. To use this code you should include xs.inc as ```#include <xs>```
