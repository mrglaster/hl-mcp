# pfn_touch Функция вызывается при косании объектов
[The Engine Module](http://amxxmodx.ru/engine/) [engine.inc](http://amxxmodx.ru/engine/engineinc/)
Честно говоря для работы с объектами есть более удобные функции других модулей, по этому данной функцией я ни когда не пользовался, но знания о ней вам все равно не повредят.  
  
Инфо из **engine.inc:**  
```
/* Called when 2 entities touch.   
 * ptr - touched entity  
 * ptd - toucher entity  
 */  
forward pfn_touch(ptr, ptd);
```
  
  
**Синтаксис:**  

public pfn_touch ( ptr, ptd )
  

    * **ptr** - id Затронутого объекта  

    * **ptd** - id Объекта  

  
  
**Тип функции:**  
Forward  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <engine>  
  
#define PLUGIN "[engine.inc] pfn_touch"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
}  
public  pfn_touch ( ptr, ptd ){  
      
    new name1[32],name2[32]  
    get_user_name(ptr,name1,31)  
    get_user_name(ptd,name2,31)  
      
    client_print(0,print_chat,"Ent1 %d Ent2 %d",ptr, ptd )  
    client_print(0,print_chat,"Ent name1: %s Ent name2: %s",name1, name2 )  
}  

```
  
  
**Описание:**  
Зайдите на сервер и просто побегайте по карте, в чат будет валиться куча записей о соприкосновении вас и объектов карты.
