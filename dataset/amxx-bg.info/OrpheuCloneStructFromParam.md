# OrpheuCloneStructFromParam
#### Syntax
```
native OrpheuStruct:OrpheuCloneStructFromParam(num)
```

#### Usage
num | ```The number of the argument. The first argument would be the number "1"```
---|---
#### Description
```
Creates a struct equal to one received as an argument
```

#### Return
```
A handler to the struct
```


This code is a part of orpheu.inc. To use this code you should include orpheu.inc as ```#include <orpheu>```
