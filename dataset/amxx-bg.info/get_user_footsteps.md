# get_user_footsteps
#### Syntax
```
native get_user_footsteps(index);
```

#### Usage
index | ```Client index```
---|---
#### Description
```
Tells whether a player has silent footsteps enabled.
```

#### Return
```
1 if silent footsteps are enabled, 0 if not
```

#### Error
```
If player is not connected or not within the range
of 1 to MaxClients.
```


This code is a part of fun.inc. To use this code you should include fun.inc as ```#include <fun>```
