# csgor_set_user_rank
#### Syntax
```
native csgor_set_user_rank(id, rangid);
```

#### Usage
id | ```Player index.```
---|---
rankid | ```A valid Rank ID.```
#### Description
```
Sets a player's Rank ID. Set the appropriate amount of kills.
```

#### Return
```
1 on success, -1 otherwise.
```


This code is a part of csgo_remake.inc. To use this code you should include csgo_remake.inc as ```#include <csgo_remake>```
