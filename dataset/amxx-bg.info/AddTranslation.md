# AddTranslation
#### Syntax
```
native AddTranslation(const lang[3], TransKey:key, const phrase[]);
```

#### Usage
lang | ```Two-letter language name```
---|---
key | ```Language key```
phrase | ```Translated text```
#### Description
```
Adds a new translation.
```

#### Return
```
This function has no return value.
```


This code is a part of lang.inc. To use this code you should include lang.inc as ```#include <lang>```
