# zp_fw_deathmatch_respawn_pre
#### Syntax
```
forward zp_fw_deathmatch_respawn_pre(id)
```

#### Usage
id | ```Player index.```
---|---
#### Description
```
Called before the deathmatch module respawns a player.
Returning PLUGIN_HANDLED will block respawn.
```


This code is a part of zp50_deathmatch.inc. To use this code you should include zp50_deathmatch.inc as ```#include <zp50_deathmatch>```


  
  

Warning! This is an external include! It does not come bundled with AMX Mod X "out of the box" and requires additional installation. The include comes bundled with the Zombie Plague 5.0 mod for Counter-Strike 1.6 and will not work with other games (Half-Life, DoD, etc.) or without the Zombie Plague 5.0 mod installed.
