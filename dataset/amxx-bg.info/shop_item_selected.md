# shop_item_selected
#### Syntax
```
forward shop_item_selected(id, itemid)
```

#### Usage
id | ```Индекс на играча```
---|---
itemid | ```Индекс на итема```
#### Description
```
Извиква се при избиране на итем от shop менюто
```

#### Note
```
Върнете стойност FORWARD_CONTINUE, за да блокирате купуването на даден итем.
```
