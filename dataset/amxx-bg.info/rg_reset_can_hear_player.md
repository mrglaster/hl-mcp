# rg_reset_can_hear_player
#### Syntax
```
native rg_reset_can_hear_player(const index);
```

#### Usage
index | ```Player id```
---|---
#### Description
```
Reset if player can hear another player.
```

#### Return
```
This function has no return value.
```
