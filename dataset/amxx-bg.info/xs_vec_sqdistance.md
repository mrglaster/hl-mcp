# xs_vec_sqdistance
#### Syntax
```
stock Float:xs_vec_sqdistance(const Float:vec1[], const Float:vec2[])
```

#### Usage
vec1 | ```First vector.```
---|---
vec2 | ```Second vector.```
#### Description
```
Computes the square of the distance between two vectors (points).
This is faster than the distance.
```

#### Return
```
The square of the distance between two vectors.
```


This code is a part of xs.inc. To use this code you should include xs.inc as ```#include <xs>```
