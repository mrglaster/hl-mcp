# zv_register_extra_item
#### Syntax
```
native zv_register_extra_item(const name[], const discription[], cost, team)
```

#### Usage
name | ```Extra item name.  @param discription    Extra item discription (simmilar like zombie class discription).  @param cost           How much ammo packs will cost this item for player.  @param team           Teams allowed to buy this item. (0 for all teams)  @return               Extra item id.```
---|---
#### Description
```
Registers extra items on this plugin for VIPs (Only if MODE 2 is active)
```


This code is a part of zmvip.inc. To use this code you should include zmvip.inc as ```#include <zmvip>```


  
  

Warning! This is an external include! It does not come bundled with AMX Mod X "out of the box" and requires additional installation. Use it only if you are absolutely certain that you need it.
