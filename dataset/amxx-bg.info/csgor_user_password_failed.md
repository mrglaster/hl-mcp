# csgor_user_password_failed
#### Syntax
```
forward csgor_user_password_failed(id, iFail);
```

#### Usage
id | ```Player index.```
---|---
fails | ```How many times player entered the wrong password.```
#### Description
```
Called when a player is entering a wrong password.
```

#### Return
```
This forward ignores the return value.
```


This code is a part of csgo_remake.inc. To use this code you should include csgo_remake.inc as ```#include <csgo_remake>```
