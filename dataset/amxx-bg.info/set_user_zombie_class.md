# set_user_zombie_class
#### Syntax
```
native set_user_zombie_class(id, classid)
```

#### Usage
id | ```Индекс на играча```
---|---
classid | ```Индекс на зомби класа (-1 - Classic Zombie)```

```
Задава индекса на зомби класа на играча
```

#### Return
```
This function has no return value.
```
