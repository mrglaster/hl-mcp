# csgor_set_user_skins
#### Syntax
```
native csgor_set_user_skins(id, skinid, amount);
```

#### Usage
id | ```Player index.```
---|---
skinid | ```A valid Skin ID.```
amount | ```A valid amount of skins.```
#### Description
```
Sets a player's skins (by specific Skin ID) to desired amount.
```

#### Return
```
1 on success, -1 otherwise.
```


This code is a part of csgo_remake.inc. To use this code you should include csgo_remake.inc as ```#include <csgo_remake>```
