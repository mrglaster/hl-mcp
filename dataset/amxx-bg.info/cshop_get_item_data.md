# cshop_get_item_data
#### Syntax
```
native any:cshop_get_item_data(item, any:type, buffer[] = "", len = 0)
```

#### Usage
item | ```Item id.```
---|---
type | ```Data type.```
buffer | ```Buffer to store string data in.```
buffer | ```Buffer length.```
#### Description
```
Returns item data.
```

#### Return
```
Integer or float type data. -1 on failure.
```


This code is a part of customshop.inc. To use this code you should include customshop.inc as ```#include <customshop>```
