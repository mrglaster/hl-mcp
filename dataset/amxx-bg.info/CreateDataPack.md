# CreateDataPack
#### Syntax
```
native DataPack:CreateDataPack();
```

#### Description
```
Creates a new datapack.
```

#### Return
```
New datapack handle, which must be freed via DestroyDataPack().
```


This code is a part of datapack.inc. To use this code you should include datapack.inc as ```#include <datapack>```
