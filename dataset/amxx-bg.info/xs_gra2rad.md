# xs_gra2rad
#### Syntax
```
stock Float:xs_gra2rad(Float:x)
```

#### Usage
x | ```Input gradians.```
---|---
#### Description
```
Converts gradians to radians.
```

#### Return
```
Gradians converted to radians.
```


This code is a part of xs.inc. To use this code you should include xs.inc as ```#include <xs>```
