# rg_set_user_ammo
#### Syntax
```
native rg_set_user_ammo(const index, WeaponIdType:weapon, amount);
```

#### Usage
index | ```Client index```
---|---
weapon | ```Weapon id```
amount | ```New clip ammo amount to set```
#### Description
```
Sets the amount of clip ammo for a specific weapon.
```

#### Return
```
This function has no return value.
```
