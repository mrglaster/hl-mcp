# Урок 26. Какой редактор выбрать для написания плагинов для AMX Mod X?
[Уроки по скриптингу для начинающих](http://amxxmodx.ru/lessons-on-pawn/)
Начинающие скриптеры рано или поздно задают вопрос, какой редактор выбрать, какой из них лучше, в чем преимущества и недостатки того или иного продукта. После прочтения статьи у Вас появится информация по данным вопросам и Вы сможете самостоятельно сделать выбор.  
  
Для начала разберём, что из себя представляет исходный код плагина. Это текстовый файл с расширением sma, который может быть представлен в кодировках ANSI или UTF-8 (без BOM). Кодировка UTF-8 (без BOM) используется для корректного отображения русскоязычных символов.  
  
Теперь зная, что исходный код - это обычный текст, можно сделать вывод что для его редактирования подойдет практически любой текстовый редактор, умеющий работать с одной из кодировок. Для написания плагинов подойдет даже программа Блокнот. Это Вы должны иметь ввиду, на тот случай, если не окажется под рукой более удобного редактора.  
  
Выбор текстовых редакторов очень велик, но большинство скриптеров пользуются одной из 2-х самых популярных программ. Первая программа это AMXX-Studio, вторая - Notepad++ AlliedModders Edition.  
  
**AMXX-Studio** - это редактор, написанный разработчиками AMX Mod X.  
  
**Notepad++ AlliedModders Edition** - это специальная версия Notepad++, настроенная для работы со скриптами на языке Pawn.  
  
Рамки статьи не позволяют наглядно продемонстрировать основные возможности этих редакторов, поэтому рекомендуются к просмотру видео-приложения.  
  
Видео-приложение 1 - AMXX-Studio  
  
  
Видео-приложение 2 - Notepad++ AlliedModders Edition  
  
  
Как Вы наверно заметили, каждый редактор имеет сильные и слабые стороны, но оба являются полноценным средством для создания плагинов. Рекомендуется попробовать обе программы и исходя из личных предпочтений сделать выбор в пользу одной.
