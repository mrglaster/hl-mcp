# VelocityByAim
#### Syntax
```
stock VelocityByAim(iIndex,iVelocity, Float:vRetValue[3])
```

#### Description
```
Gives you a velocity in the direction a player is looking, iVelocity is the multiplier.
```


This code is a part of engine_stocks.inc. To use this code you should include engine_stocks.inc as ```#include <engine_stocks>```
