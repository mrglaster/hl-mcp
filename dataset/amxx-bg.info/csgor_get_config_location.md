# csgor_get_config_location
#### Syntax
```
native csgor_get_config_location(szLocation[64]);
```

#### Usage
szLocation | ```Buffer to copy config location```
---|---
#### Description
```
Returns location of configuration file.
```

#### Return
```
This function has no return value.
```


This code is a part of csgo_remake.inc. To use this code you should include csgo_remake.inc as ```#include <csgo_remake>```
