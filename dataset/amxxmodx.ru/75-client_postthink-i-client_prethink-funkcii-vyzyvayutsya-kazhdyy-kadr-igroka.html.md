# client_PostThink и client_PreThink Функции вызываются каждый кадр игрока
[The Engine Module](http://amxxmodx.ru/engine/) [engine.inc](http://amxxmodx.ru/engine/engineinc/)
Функции **client_PreThink** и **client_PostThink** вызываются каждый кадр игрока, если FPS у игрока 30 - то будет вызвано 30 раз в секунду, если 100 то сто раз. Отличаются функции только тем что одна вызывается до кадра, вторая после. Обращаю сразу же ваше внимание, что данные функци Очень сильно нагружают сервер. не используйте их без крайней необходимости.  
  
Инфо из **engine.inc:**  
```
/* Forward for PreThink()/PostThink() on a player. */  
forward client_PreThink(id);  
forward client_PostThink(id);
```
  
  
**Синтаксис:**  

public client_PostThink ( id )  
public client_PreThink ( id )  

  

    * **id** - Индекс игрока  

  
  
**Пример:**  
```
  
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <engine>  
  
#define PLUGIN "PreThink"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
new counter  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
}  
public client_PreThink(id)  
{  
    counter++  
    client_print(id, print_console, "Counter: %d",counter)  
}
```
  
  
**Описание:**  
Как только вы зайдете на сервер и откроете консоль, вы сразу увидите бегущие строки, и чем больше у вас FPS тем быстрее они будут бежать. Для наглядности там в каждой строке выводится число, к которому каждый раз прибавляем 1.  
Можете попробовать изменять значение вашего FPS со 100 до 20 и вы сразу же заметите разницу.
