[Core](http://127.0.0.1:8000/content/index.htm) (amxmodx.inc) 
Description
get_flags - Converts a sum of bits into a string. 
Syntax
get_flags ( flags, output[], len ) 
Type
Native 
Notes
This is the opposite of [read_flags](http://127.0.0.1:8000/content/read_flags.htm).   
  
The result is stored in output for a maximum length of len characters.
