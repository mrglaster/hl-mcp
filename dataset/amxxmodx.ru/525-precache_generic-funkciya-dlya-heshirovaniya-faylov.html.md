# precache_generic Функция для хеширования файлов
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Для хеширования моделей и звуков есть отдельные функции, для остальных файлов есть функция precache_generic.  
  
Инфо из **amxmodx.inc:**  
```
/* Precaches any file. */  
native precache_generic(const szFile[]);
```
  
  
**Синтаксис:**  

precache_generic(const szFile[])
  

    * **const szFile[])** - кешируемый файл  

  
  
**Тип функции:**  
Native  
  
**Пример и описание:**  
Смотри функции [precache_model](http://amxxmodx.ru/core/amxmodxinc/334-precache_model-funkciya-predvaritelno-keshiruet-fayl-modelisprayta.html) и [precache_sound](http://amxxmodx.ru/core/amxmodxinc/333-precache_sound-funkciya-predvaritelno-keshiruet-zvukovoy-fayl.html)
