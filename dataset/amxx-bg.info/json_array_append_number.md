# json_array_append_number
#### Syntax
```
native bool:json_array_append_number(JSON:array, number);
```

#### Usage
array | ```Array handle```
---|---
number | ```Number to set```
#### Description
```
Appends a number in the array.
```

#### Return
```
True if succeed, false otherwise
```

#### Error
```
If passed handle is not a valid array
```


This code is a part of json.inc. To use this code you should include json.inc as ```#include <json>```
