# Chicken shooting gallery или Куриный тир
[Готовые плагины](http://amxxmodx.ru/amxx_plugins/)
[![Chicken shooting gallery или Куриный тир](http://amxxmodx.ru/uploads/posts/2013-04/thumbs/1365968590_cs_chicken_b20004.jpg)](http://amxxmodx.ru/uploads/posts/2013-04/1365968590_cs_chicken_b20004.jpg)  
**Используемые модули:**  

    * amxmodx  

    * fakemeta  

    * hamsandwich  

  
  
**Команды:**  
say /tir - открытие меню тира  
say /menu - открытие меню тира  
say /stop - открытие меню тира   
  
Если игрок вне тира - меню тира, если в тире только 1 вариант выхода из тира.  
  
**Типы игры ( игрок выбирает сам):**  
1. Как можно больше убить кур за 1 минуту ( в одиночку )  
2. Без ограничения времени ( в одиночку)  
3. Мультиплеер  
  
**Каждый игрок может выбирать:**  
1 Уровень сложности  
2 Оружие как в GunGame или получать баксы за убийства и покупать оружие  
3 Жизни курицы ( только в одиночной игре)  
  
На карте 4 тира, 3 для одиночной игры 1 для нескольких человек.  
Двери закрываются автоматически, если игрок в тире.  
В тиры пройти нельзя, происходит телепортация при старте туда при финише обратно.  
Курицы автоматически ускоряются при убийстве и замедляются если улетают.  
  
**Настройка:**  
Работает только на карте **cs_chicken_b2** (карта в комплекте)  
Квар amx_chiken_gg_kills - Для режима GunGame сколько кур убить до следующего уровня ( по умолчанию 3)
