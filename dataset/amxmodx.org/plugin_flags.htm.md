Description
plugin_flags - Returns plugin flags 
Syntax
plugin_flags ( [ hdr=0 ] ) 
Type
Native 
Notes
Returns the internal flags set on the called plugin's state. If hdr is 1, it will return the pcode flags rather than state flags.   
  
These flags are defined in amxconst.inc
