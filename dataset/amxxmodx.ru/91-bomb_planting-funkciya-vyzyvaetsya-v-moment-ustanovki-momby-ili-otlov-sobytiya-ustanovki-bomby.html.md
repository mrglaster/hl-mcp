# bomb_planting Функция вызывается в момент установки момбы или отлов события установки бомбы
[The CSX Module](http://amxxmodx.ru/csx/) [csx.inc](http://amxxmodx.ru/csx/csxinc/)
Функцию **bomb_planting** , я встречал только в стандартном плагине **miscstats** , хотя планирую в некоторые своих плагинах использовать данную функцию. Ах да, о функции, она вызывается каждый раз когда кто то начинает устанавливать бомбу. Именно устанавливать, для события успешной установки есть функция **bomb_planted**.  
  
Инфо из **csx.inc:**  
```
forward bomb_planting(planter);
```
  
  
**Синтаксис:**  

public bomb_planting ( planter )
  

    * **planter** - id игрока ставящего бомбу  

  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <csx>  
  
#define PLUGIN "Event bomb planting"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
}  
public bomb_planting(id){  
  
    new name[32]  
    get_user_name(id,name,31)  
      
    set_hudmessage(255, 127, 0, -1.0, 0.62, 0, 6.0, 6.0)  
    show_hudmessage(id, "%s planting bomb",name)  
  
}  

```
  
  
**Описание:**  
Как видно в примере, все очень просто, функция вызывается автоматически каждый раз когда кто то начинает ставить бомбу, для наглядности мы получим имя этого игрока и выведем HuD сообщение что он ставит бомбу.
