# min
#### Syntax
```
native min(value1, value2);
```

#### Usage
value1 | ```Value one```
---|---
value2 | ```Value two```
#### Description
```
Returns the smaller of two provided values.
```

#### Return
```
Smaller of the two values
```


This code is a part of core.inc. To use this code you should include core.inc as ```#include <core>```
