# precache_get_item
#### Syntax
```
native precache_get_item(iNum, szItem[], iLen, iType)
```

#### Usage
iNum | ```Index of resource in array.```
---|---
szItem[] | ```Buffer to copy resource name.```
iLen | ```Lenght of buffer.```
iType | ```Type of the resource. See _:Types Enum```
#### Description
```
Gets the resource name from array.
```

#### Type
```
Integer
```

#### Return
```
1 on succes, -1 on error.
```


This code is a part of precache_list.inc. To use this code you should include precache_list.inc as ```#include <precache_list>```
