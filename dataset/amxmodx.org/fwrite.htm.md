Description
fwrite - Writes to a file handle. 
Syntax
fwrite ( file, const str[], ... ) 
Type
Native 
Notes
Writes a string of data to a file handle. The string can be formatted in the style of [format](http://127.0.0.1:8000/content/format.htm).   
  
This function is deprecated as of AMX Mod X Version 1.1.
