# set_user_footsteps
#### Syntax
```
native set_user_footsteps(id, set = 1);
```

#### Usage
index | ```Client index```
---|---
set | ```1 if player should have silent footsteps, 0 otherwise```
#### Description
```
Enables or disables player's silent footsteps.
```

#### Return
```
This function has no return value.
```

#### Error
```
If player is not connected or not within the range
of 1 to MaxClients.
```


This code is a part of fun.inc. To use this code you should include fun.inc as ```#include <fun>```
