# str_to_float
#### Syntax
```
native Float:str_to_float(const string[]);
```

#### Usage
string | ```String to convert to a foat.```
---|---
#### Description
```
Converts a string to a floating point number.
```

#### Return
```
Floating point result, or 0.0 on error.
```


This code is a part of string.inc. To use this code you should include string.inc as ```#include <string>```
