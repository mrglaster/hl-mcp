# OrpheuGetLibraryAddress
#### Syntax
```
native OrpheuGetLibraryAddress(const libraryName[])
```

#### Usage
libraryName | ```The name of the library. For the mod library (cstrike,czero,etc) use "mod"```
---|---
#### Description
```
Gets the adress of a library
```

#### Return
```
The address
```
