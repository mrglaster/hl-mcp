# GetFileTime
#### Syntax
```
native GetFileTime(const file[], FileTimeType:tmode);
```

#### Usage
file | ```File name```
---|---
tmode | ```Time mode, see FileTime_* constants```
#### Description
```
Returns a file timestamp as a unix timestamp.
```

#### Return
```
Returns a file timestamp as a unix timestamp
```


This code is a part of file.inc. To use this code you should include file.inc as ```#include <file>```
