# set_user_maxspeed
#### Syntax
```
native set_user_maxspeed(index, Float:speed = -1.0);
```

#### Usage
index | ```Client index```
---|---
speed | ```The maximum speed player will be able to run at```
#### Description
```
Sets player's maximum movement speed.
```

#### Return
```
This function has no return value.
```

#### Error
```
If player is not connected or not within the range
of 1 to MaxClients.
```


This code is a part of fun.inc. To use this code you should include fun.inc as ```#include <fun>```
