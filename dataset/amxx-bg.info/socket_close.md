# socket_close
#### Syntax
```
native socket_close(_socket);
```

#### Usage
_socket | ```Socket descriptor```
---|---
#### Description
```
Closes a socket.
```

#### Return
```
1 on success
0 on failure
```


This code is a part of sockets.inc. To use this code you should include sockets.inc as ```#include <sockets>```
