# set_cvar_float Функция изменяет значения кваров с плавающей точкой
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Очередная функция для изменения значений кваров на сервер, на этот раз функция **set_cvar_float** , изменяющая квары с дробным значением.  
  
Инфо из **amxmodx.inc:**  
```
/* Sets a cvar to given float. */  
native set_cvar_float(const cvar[],Float:value);
```
  
  
**Синтаксис:**  

[/code]  
[list]  
[*][b]const cvar[][/b] - Название квара  
[*][b][/b] - Значение или переменная с новым значением  
[/list]  
  
[b]Тип функции:[/b]  
Native   
  
[b]Пример:[/b]  
[code=pawn]/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
  
#define PLUGIN "[amxmodx.inc] set_cvar_float "  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
  
public plugin_init() {  
register_plugin(PLUGIN, VERSION, AUTHOR)  
  
register_cvar("amx_my_float_cvar","5.5")  
  
register_srvcmd("set_my_float_cvar","f_set_cvar_float")  
register_srvcmd("get_my_float_cvar","cvars")  
  
}  
public f_set_cvar_float(){  
set_cvar_float("amx_my_float_cvar",100.9)  
  
cvars()  
}  
public cvars(){  
  
new Float:my_cvar = get_cvar_float("amx_my_float_cvar")  
server_print("[1]My float cvar = %.1f",my_cvar)  
server_print("[2]My float cvar = %.1f",get_cvar_float("amx_my_float_cvar"))  
  
}
  
  
**Описание:**  
Смотри другие функции **set_cvar_*** все тоже самое.
