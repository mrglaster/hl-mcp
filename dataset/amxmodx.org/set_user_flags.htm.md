[Core](http://127.0.0.1:8000/content/index.htm) (amxmodx.inc) 
Description
set_user_flags - Sets flags for player. 
Syntax
set_user_flags ( index,flags=-1,id=0 ) 
Type
Native 
Notes
Set flags to -1 if you want to clear all flags. You can use different settings by changing the id, which is from range 0 - 31.
