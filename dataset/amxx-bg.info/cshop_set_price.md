# cshop_set_price
#### Syntax
```
forward cshop_set_price(id, item, price)
```

#### Usage
id | ```Player's index.```
---|---
ite, | ```Item's id.```
price | ```Item's price.```
#### Description
```
Called before a menu price is set.
```

#### Return
```
This forward ignores the return value.
```


This code is a part of customshop.inc. To use this code you should include customshop.inc as ```#include <customshop>```
