# json_array_append_string
#### Syntax
```
native bool:json_array_append_string(JSON:array, const string[]);
```

#### Usage
array | ```Array handle```
---|---
string | ```String to copy```
#### Description
```
Appends string data in the array.
```

#### Return
```
True if succeed, false otherwise
```

#### Error
```
If passed handle is not a valid array
```


This code is a part of json.inc. To use this code you should include json.inc as ```#include <json>```
