# FClassnameIs
#### Syntax
```
native bool:FClassnameIs(const entityIndex, const className[]);
```

#### Description
```
Compares the entity to a specified classname.
```

#### Note
```
This native also checks the validity of an entity.
```

#### Return
```
true/false
```
