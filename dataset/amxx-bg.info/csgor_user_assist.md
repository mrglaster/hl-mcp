# csgor_user_assist
#### Syntax
```
forward csgor_user_assist(id, killer, victim, head);
```

#### Usage
id | ```Assist index.```
---|---
killer | ```Killer index.```
victim | ```Victim index.```
head | ```1 if HeadShot, 0 otherwise.```
#### Description
```
Called when a player make assist.
```

#### Return
```
This forward ignores the return value.
```


This code is a part of csgo_remake.inc. To use this code you should include csgo_remake.inc as ```#include <csgo_remake>```


  
  

Warning! This is an external include! It does not come bundled with AMX Mod X "out of the box" and requires additional installation. Use it only if you are absolutely certain that you need it.
