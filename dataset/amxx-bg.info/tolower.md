# tolower
#### Syntax
```
native tolower(c);
```

#### Usage
c | ```Character to convert```
---|---
#### Description
```
Converts a character to lowercase.
```

#### Note
```
This is not UTF8 or locale-safe.
```

#### Return
```
Converted character
```


This code is a part of core.inc. To use this code you should include core.inc as ```#include <core>```
