# cshop_get_prefix
#### Syntax
```
native cshop_get_prefix(prefix[], len)
```

#### Usage
prefix | ```Buffer.```
---|---
len | ```Max buffer length.```

```
Gets the plugin prefix.
```

#### Return

This function has no return value.
