# rmdir
#### Syntax
```
native rmdir(const path[]);
```

#### Usage
path | ```Path to the directory```
---|---
#### Description
```
Removes a directory.
```

#### Note
```
On most Operating Systems you cannot remove a directory which has files inside it.
```

#### Return
```
1 on success, 0 otherwise
```
