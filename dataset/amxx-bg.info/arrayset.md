# arrayset
#### Syntax
```
native arrayset(any:array[], any:value, size);
```

#### Usage
array | ```Array to modify```
---|---
value | ```Value to set each element to```
size | ```Size of array```
#### Description
```
Sets all elements of array to a specified value.
```

#### Return
```
This function has no return value.
```
