# cshop_get_string
#### Syntax
```
native cshop_get_string(item, setting[], value[], len)
```

#### Usage
item | ```Item id.```
---|---
setting | ```Setting's name.```
value | ```Buffer for the value.```
len | ```Buffer length.```
#### Description
```
Returns a string value from a specific item's setting.
```

#### Return
```
This function has no return value.
```


This code is a part of customshop.inc. To use this code you should include customshop.inc as ```#include <customshop>```
