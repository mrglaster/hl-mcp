# set_pcvar_string Функция изменяет значения квара (работает со строками)
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Очередная функция изменяющая значения кваров, на этот раз функция изменяет текстовые значения кваров, например имена для ботов.  
Читайте так же о функции [**get_pcvar_string**](http://amxxmodx.ru/engine/go.php?url=aHR0cDovL2FteHhtb2R4LnJ1L2NvcmUvYW14bW9keGluYy8zMzUtZ2V0X3BjdmFyX3N0cmluZy1mdW5rY2l5YS1wb2x1Y2hhZXQtc2ltdm9sbm9lLXpuYWNoZW5pZS1jdmFyLmh0bWw%3D).  
  
Инфо из **amxmodx.inc:**  
```
  
native set_pcvar_string(pcvar, const string[]);
```
  
  
**Синтаксис:**  
```
set_pcvar_string(pcvar, const string[])
```
  

    * **pcvar** - указатель на квар  

    * **const string[]** - значение, которое будет записано в квар  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
  
#define PLUGIN "[amxmodx.inc] set_pcvar_string"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
new pcvar  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
    //Получение указателя на квар  
    pcvar = register_cvar("test_pcvar","Mega Text")  
      
    //Массив для получаемого значения  
    new string[32]  
    //Получаем значение квара  
    get_pcvar_string(pcvar,string,31)  
      
    //Вывод в консоль сервера полученного значения      
    server_print("^n Pcvar:%s ^n",string)  
      
    register_concmd("set_cvar","set_cvar")  
      
}  
public set_cvar(){  
      
    set_pcvar_string(pcvar,"New Mega Text")  
      
    new string[32]  
    get_pcvar_string(pcvar,string,31)  
    server_print("^n Pcvar:%s ^n",string)  
}
```
