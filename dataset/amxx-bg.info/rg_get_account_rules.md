# rg_get_account_rules
#### Syntax
```
native rg_get_account_rules(const RewardRules:rules_index);
```

#### Usage
rules_index | ```Look at the enum with name RewardRules```
---|---
#### Description
```
Gets the specified reward rule's money amount.
```

#### Return
```
The amount of reward
```
