# nvault_set Функция устанавливает значение с изменением временной метки
[The nVault Module](http://amxxmodx.ru/nvaultinc/)
После открытия файла с помощью функции [**nvault_open**](http://amxxmodx.ru/nvaultinc/402-nvault_open-funkciya-otkryvaet-fayl-s-binarnymi-dannymi.html) хочется начать работать с данными, но их там нет. Давайте поместим их туда с помощью функции **nvault_set  
**  
  
Инфо из **nvault.inc:**  
```
/* Sets a vault value (with current timestamp) */  
native nvault_set(vault, const key[], const value[]);
```
  
  
**Синтаксис:**  

nvault_set ( vault, const key[], const value[] )
  

    * **vault** - Идентификатор открытого файла  

    * **const key[]** - Значение ключа.  

    * **const value[]** - Записываемая переменная.  

Функция так же обновляет временную метку timestamp  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <nvault>  
  
#define PLUGIN "[nvault.inc] nvault_set"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
    new hvault = nvault_open("mega_file")  
  
    new s_ID[] = "zero"  
          
    nvault_set(hvault, s_ID, "hello world")  
      
}
```
