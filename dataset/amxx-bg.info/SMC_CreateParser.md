# SMC_CreateParser
#### Syntax
```
native SMCParser:SMC_CreateParser();
```

#### Description
```
Creates a new SMC parser.
This is used to set parse hooks.
```

#### Return
```
A new handle to an SMC Parse structure.
```


This code is a part of textparse_smc.inc. To use this code you should include textparse_smc.inc as ```#include <textparse_smc>```
