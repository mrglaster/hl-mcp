# xs_is_2power
#### Syntax
```
stock xs_is_2power(x)
```

#### Usage
x | ```Number to check.```
---|---
#### Description
```
Checks if the number is a power of 2.
```

#### Return
```
1 if it is a power of 2, 0 otherwise.
```


This code is a part of xs.inc. To use this code you should include xs.inc as ```#include <xs>```
