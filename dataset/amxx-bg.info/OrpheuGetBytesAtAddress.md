# OrpheuGetBytesAtAddress
#### Syntax
```
native OrpheuGetBytesAtAddress(address,bytes[],count)
```

#### Usage
address | ```The direct address in memory```
---|---
bytes | ```An array to hold the bytes```
count | ```The number of bytes to get```
#### Description
```
Gets the bytes found at an address
```


This code is a part of orpheu_advanced.inc. To use this code you should include orpheu_advanced.inc as ```#include <orpheu_advanced>```
