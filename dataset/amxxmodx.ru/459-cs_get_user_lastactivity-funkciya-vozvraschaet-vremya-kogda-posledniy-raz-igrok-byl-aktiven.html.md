# cs_get_user_lastactivity Функция возвращает время когда последний раз игрок был активен
[CStrike](http://amxxmodx.ru/cstrike/)
С помощью данной функции можно организовать свой палгин afk kicker.  
  
Инфо из **cstrike.inc:**  
```
/* Gets or sets the user's last activity time.  This is the time that CS's internal afk kicker  
 * checks to see who has been afk too long.  
 */  
native Float:cs_get_user_lastactivity(index);
```
  
  
**Синтаксис:**  

cs_get_user_lastactivity(index)
  

    * **index** - Индекс игрока  

Функция возвращает дробное значение когда игрок последний раз был активен (относительно старта сервера)  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <cstrike>  
  
#define PLUGIN "[http://amxxmodx.ru] cs_get_user_lastactivity"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
  
    register_clcmd("say la","test_func")  
  
}  
  
public test_func(id){  
      
    client_print(0,print_chat,"My last activiti  [%f] systime[%f]",cs_get_user_lastactivity(id),get_gametime())  
      
}
```
  
  
**Описание:**  
Для наглядности в одну строку с вашей последней активностью выводится системное время.
