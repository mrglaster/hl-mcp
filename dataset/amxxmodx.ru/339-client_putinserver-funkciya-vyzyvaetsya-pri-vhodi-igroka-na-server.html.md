# client_putinserver Функция вызывается при входе игрока на сервер
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Форвард функция **client_putinserver** вызывается каждый раз когда игрок заходит на сервер ( именно в тот момент когда он начинает видеть карту и первое MODT окно)  
  
Инфо из **amxmodx.inc:**  
```
/* Called when client is entering to a game. */  
forward client_putinserver(id);
```
  
  
**Синтаксис:**  

client_putinserver(id)
  

    * **id** - Айди зашедшего игрока  

  
  
**Тип функции:**  
Forward  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
  
  
#define PLUGIN "[amxmodx.inc] client_authorized"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
}  
public client_putinserver(id){  
      
    server_print("^n^nclient_putinserver^n^n")  
      
}
```
  
  
**Описание:**  
Обращаю внимание на то, что хоть функция вызывается в момент когда игрок зашел на сервер, если в этой функции вывести сообщение игроку, то он все равно его не увидит.
