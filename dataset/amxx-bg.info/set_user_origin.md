# set_user_origin
#### Syntax
```
native set_user_origin(index, const origin[3]);
```

#### Usage
index | ```Client index```
---|---
origin | ```Origin to move a player to```
#### Description
```
Moves a player to the given origin.
```

#### Return
```
This function has no return value.
```

#### Error
```
If player is not connected or not within the range
of 1 to MaxClients.
```


This code is a part of fun.inc. To use this code you should include fun.inc as ```#include <fun>```
