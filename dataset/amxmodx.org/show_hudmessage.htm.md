how_hudmessage
[Core](http://127.0.0.1:8000/content/index.htm) (amxmodx.inc) 
Description
show_hudmessage - Displays a HUD message to a player 
Syntax
show_hudmessage ( index, message[], ... ) 
Type
Native 
Notes
index is a player index from 1 to 32.   
  
message can be formatted according to [format](http://127.0.0.1:8000/content/format.htm).   
Remeber to use the [set_hudmessage](http://127.0.0.1:8000/content/set_hudmessage.htm) function first, to control the color/pos/effects.
