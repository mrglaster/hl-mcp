# Список функций для работы с файлами из file.inc
[Ядро AMXx](http://amxxmodx.ru/core/) [file.inc](http://amxxmodx.ru/core/fileinc/)
Полный список функций для работы с файлами:  
[**read_dir**](http://amxxmodx.ru/core/fileinc/251-read_dir-funkciya-chitaet-direktoriyu.html) читает директорию  
[**read_file**](http://amxxmodx.ru/core/fileinc/260-read_file-funkciya-chitaet-stroku-iz-fayla.html) читает строку из файла  
[**write_file**](http://amxxmodx.ru/core/fileinc/259-write_file-funkciya-zapisyvayet-stroku-v-fayl.html) записывает строку в файл.  
[**delete_file**](http://amxxmodx.ru/core/fileinc/268-delete_file-funkciya-udalyaet-fayl.html) удаляет файл  
[**file_exists**](http://amxxmodx.ru/core/fileinc/249-file_exists-funkciya-proveryaet-suschestvovanie-fayla.html) проверяет существует файл или нет  
[**rename_file**](http://amxxmodx.ru/core/fileinc/258-rename_file-funkciya-pereimenovyvaet-fayl.html) переименовывает файл  
[**dir_exists**](http://amxxmodx.ru/core/fileinc/250-dir_exists-funkciya-proveryaet-suschestvuet-direktoriya-ili-net.html) проверяет существует директория или нет  
[**file_size**](http://amxxmodx.ru/core/fileinc/265-file_size-funkciya-poluchaet-informaciyu-o-razmere-fayla-i-kolichestve-strok.html) получает информацию о размере файла и количестве строк  
[**fopen**](http://amxxmodx.ru/core/fileinc/297-fopen-funkciya-otkrytiya-fayla-i-vozvrata-ego-diskriptora.html) открытия,создание файла и возврата его дескриптора.  
[**fclose**](http://amxxmodx.ru/core/fileinc/310-fclose-funkciya-dlya-zakrytiya-fayla-po-identifikatoru.html) для закрытия файла по идентификатору  
[**fread**](http://amxxmodx.ru/core/fileinc/491-fread.html)  
[**fread_blocks**](http://amxxmodx.ru/core/fileinc/508-fread_blocks-funkciya-schityvaet-blok-dannyh-iz-fayla.html) считывает блок данных из файла  
[**fread_raw**](http://amxxmodx.ru/core/fileinc/509-fread_raw-funkciya-chteniya-bloka-dannyh-v-formate-raw.html) читает блок данных в формате raw  
[**fwrite**](http://amxxmodx.ru/core/fileinc/510-fwrite-funkciya-dlya-zapisi-elementa-v-fayl.html) записывает данные в файл  
[**fwrite_blocks**](http://amxxmodx.ru/core/fileinc/520-fwrite_blocks-funkciya-zapisyvat-blok-dannyh-v-fayl.html) Функция записывает блок данных в файл.  
**fwrite_raw**  
[**feof**](http://amxxmodx.ru/core/fileinc/378-feof-funkciya-proveryaet-dostignut-li-konec-fayla.html) проверяет достигнут конец файла или нет  
[**fgets**](http://amxxmodx.ru/core/fileinc/314-fgets-funkciya-schityvaet-odnu-stroku-iz-fayla.html) считывает одну строку из файла  
[**fputs**](http://amxxmodx.ru/core/fileinc/313-fputs-funkciya-dlya-zapisi-stroki-v-fayl.html) для записи строки в файл  
[**fprintf**](http://amxxmodx.ru/core/fileinc/492-fprintf-funkciya-zapisyvaet-stroku-v-fayl-s-vozmozhnostyu-formatirovaniya-stroki.html) записывает строку в файл с возможностью форматирования строки  
[**fseek**](http://amxxmodx.ru/core/fileinc/493-fseek-funkciya-dlya-ustanovki-tekuschey-pozicii-kursora-v-fayle.html) устанавливает позицию курсора в файле  
[**ftell**](http://amxxmodx.ru/core/fileinc/494-ftell-funkciya-vozvraschaet-tekuschuyu-poziciyu-kursora-v-fayle.html) возвращает текущую позицию курсора в файле  
[**fgetc**](http://amxxmodx.ru/core/fileinc/314-fgets-funkciya-schityvaet-odnu-stroku-iz-fayla.html) считывает одну строку из файла  
[**fputc**](http://amxxmodx.ru/core/fileinc/313-fputs-funkciya-dlya-zapisi-stroki-v-fayl.html) для записи строки в файл  
[**filesize**](http://amxxmodx.ru/core/fileinc/489-filesize-funkciya-poluchaet-razmer-fayla.html) получает размер файла  
[**rmdir**](http://amxxmodx.ru/core/fileinc/488-rmdir-funkciya-udalyaet-katalog-papku.html) удаляет каталог (папку)  
[**mkdir**](http://amxxmodx.ru/core/fileinc/266-mkdir-funkciya-sozdaet-papku-na-servere.html) создает папку на сервере  
[**unlink**](http://amxxmodx.ru/core/fileinc/487-unlink-funkciya-udalyaet-fayl.html) удаляет файл  
[**open_dir**](http://amxxmodx.ru/core/fileinc/257-funkcii-open_dirnext_file-i-close_dir-chtenie-direktoriy.html) открывает директорию  
[**next_file**](http://amxxmodx.ru/core/fileinc/257-funkcii-open_dirnext_file-i-close_dir-chtenie-direktoriy.html) берет следующий файл  
[**close_dir**](http://amxxmodx.ru/core/fileinc/257-funkcii-open_dirnext_file-i-close_dir-chtenie-direktoriy.html) закрывает директорию
