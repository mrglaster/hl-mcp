# log_message Функция добавления сообщения в стандартный лог файл HL
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
С помощью данной функции вы можете добавить необходимое сообщение в стандартный лог фай HL.  
Обратите внимание что логирование при этом должно быть включено!  
  
Инфо из **amxmodx.inc:**  
```
/* Sends message to standard HL logs. */  
native log_message(const message[],any:...);
```
  
  
**Синтаксис:**  

log_message ( const message[], [ ... ] )
  

    * **const message[],** - Собщение или подготовленный массив с сообщением  

    * **[ ... ]** - массивы и переменные для подстановки  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
  
#define PLUGIN "[amxmodx.inc] log_message"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
  
    register_concmd("write_log_to_hl_file","write_log_to_hl_file")  
}  
public write_log_to_hl_file(){  
      
    new map[32]  
    get_mapname(map,31)  
      
    log_message("TEST MESSAGE ON MAP %s",map)  
}  

```
  
  
**Описание:**  
Все так же просто как и с другими сообщениями и логированием.  
Подготавливайте сообщение заранее или в теле функции с поставлением необходимых массивов или переменных
