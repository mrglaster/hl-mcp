# rg_remove_all_items
#### Syntax
```
native rg_remove_all_items(const index, const bool:removeSuit = false);
```

#### Usage
index | ```Client index```
---|---
removeSuit | ```Remove suit```
#### Description
```
Remove all of the player's items.
```

#### Return
```
1 on success, 0 otherwise
```
