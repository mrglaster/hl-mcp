# fprintf Функция записывает строку в файл с возможностью форматирования строки
[Ядро AMXx](http://amxxmodx.ru/core/) [file.inc](http://amxxmodx.ru/core/fileinc/)
Инфо из **file.inc:**  
```
//Writes a line to the file  
native fprintf(file, const fmt[], any:...);
```
  
  
**Синтаксис:**  

fprintf(file, const fmt[], any:...)
  

    * **file** - индекс открытого файла  

    * **const fmt[]** - Строка для записи в файл   

    * **any:...** - аргументы для форматирования
