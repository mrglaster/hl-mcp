# ReadFlags
#### Syntax
```
stock ReadFlags(const flags[])
```

#### Usage
flags | ```String with flags to parse.```
---|---
#### Description
```
Converts string of flags into an integer value.
```

#### Note
```
Only chars from 'a' to 'z' are converted, others are skipped.
```

#### Return
```
Integer value represeting given flags string.
```


This code is a part of common_functions.inc. To use this code you should include common_functions.inc as ```#include <common_functions>```
