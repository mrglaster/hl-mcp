# set_user_gravity
#### Syntax
```
native set_user_gravity(index, Float:gravity = 1.0);
```

#### Usage
index | ```Client index```
---|---
gravity | ```Gravity value to set, 1.0 being normal gravity (800)```
#### Description
```
Sets player's gravity.
```

#### Return
```
This function has no return value.
```

#### Error
```
If player is not connected or not within the range
of 1 to MaxClients.
```


This code is a part of fun.inc. To use this code you should include fun.inc as ```#include <fun>```
