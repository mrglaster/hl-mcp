# get_user_noclip Функция получает включен ли режим noclip у игрока
[The Fun Module](http://amxxmodx.ru/fun/) [fun.inc](http://amxxmodx.ru/fun/funinc/)
Функция проверяет статус noclip у игрока, noclip это что то вроде живого спектатора, то есть вы сохраняете все свойства живого игрока, но можете проходить через любые предметы, словно призрак.  
  
Инфо из **fun.inc:**  
```
/* Returns 1 if noclip is set. */  
native get_user_noclip(index);
```
  
  
**Синтаксис:**  

get_user_noclip ( index )
  

    * **index** - id проверяемого игрока  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <fun>  
  
#define PLUGIN "[fun.inc] get_user_noclip"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
    register_clcmd("say /get_noclip","get_noclip")  
    register_clcmd("say /set_noclip","set_noclip")  
  
}  
public get_noclip(id){  
      
    new user_no_clip = get_user_noclip(id)  
    client_print(0,print_chat,"You no clip status:%s",user_no_clip ? "Yes" : "No")  
  
}  
public set_noclip(id){  
  
    set_user_noclip(id,1)  
    get_noclip(id)  
}
```
  
  
**Описание:**  
Команда **say /get_noclip** возвращает статус noclip для игрока.  
Команда **say /set_noclip** нужна для изменения статуса, что бы наглядно видеть как меняется результат работы функции.
