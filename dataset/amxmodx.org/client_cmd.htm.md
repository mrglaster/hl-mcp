Description
client_cmd - Executes command on a player 
Syntax
client_cmd ( index, const command[], {Float,...} ) 
Type
Native 
Notes
Index is a player index from 1 to 32. A zero index executes a command on all players.   
  
The command is formatted in the style of [format](http://127.0.0.1:8000/content/format.htm).
