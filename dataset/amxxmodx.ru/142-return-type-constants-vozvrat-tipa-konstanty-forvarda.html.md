# Return Type Constants Возврат типа константы форварда
[The Fakemeta Module](http://amxxmodx.ru/fakemeta/) [fakemeta_const.inc](http://amxxmodx.ru/fakemeta/fakemeta_constinc/)
Константы для функции **forward_return**  
  
Return Type Constants   
```
  
//For forward_return  
#define FMV_STRING    1  
#define FMV_FLOAT    2  
#define FMV_CELL        3
```
  

    * **FMV_STRING** - Возвращаемое значение строка  

    * **FMV_FLOAT** - Возвращаемое значение Дробное число ( число с плавающей точкой)  

    * **FMV_CELL** - Точно не уверен, возможно "Ячейка" то есть для передачи 0/1
