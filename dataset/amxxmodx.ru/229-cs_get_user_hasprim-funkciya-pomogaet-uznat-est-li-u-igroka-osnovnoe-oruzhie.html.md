# cs_get_user_hasprim Функция помогает узнать есть ли у игрока основное оружие
[CStrike](http://amxxmodx.ru/cstrike/)
Честно говоря не встречал использование данной функции в плагинах, но мне она показалось удобной, так ка кона возвращает наличие у игрока основного орудия. Обратите внимание на то, что она получает именно наличие оружия, а не какое в руках. то есть ели у игрока есть автомат, а в руках у него пистолет, то ответ от функции будет все равно положительным.  
  
Инфо из **cstrike.inc:**  
```
/* Returns 1 when user has a primary weapon   
OR a shield in inventory, else 0. */  
native cs_get_user_hasprim(index);
```
  
  
**Синтаксис:**  

cs_get_user_hasprim ( index )
  

    * **index** - id игрока  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <cstrike>  
  
#define PLUGIN "[cstrike.inc] cs_get_user_hasprim"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
  
    register_clcmd("say /wpn","f_my_wpn")  
      
}  
public f_my_wpn(id){  
      
    client_print(id,print_chat,"[cstrike.inc] You %s primary weapon ",cs_get_user_hasprim(id)? "have" : "dont have")  
  
}  

```
  
  
**Описание:**  
Собственно описывать здесь нечего.  
Топаем на сервер, покупаем оружие и выполняем команду **say /wpn** , и смотрим на результат. Затем выкидываем оружие и повторяем процедуру.  
Ах да, функция так же вернет 1 если у игрока есть щит.
