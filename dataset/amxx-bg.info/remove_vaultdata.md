# remove_vaultdata
#### Syntax
```
native remove_vaultdata(const key[]);
```

#### Usage
key | ```Key to remove```
---|---
#### Description
```
Removes a key and its data from the vault.
```

#### Return
```
This function has no return value.
```


This code is a part of vault.inc. To use this code you should include vault.inc as ```#include <vault>```
