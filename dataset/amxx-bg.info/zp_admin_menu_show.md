# zp_admin_menu_show
#### Syntax
```
native zp_admin_menu_show(id)
```

#### Usage
id | ```Player index.```
---|---
#### Description
```
Shows admin menu to a specific player. Commands will only
be available if player has the required admin flags.
```


This code is a part of zp50_admin_menu.inc. To use this code you should include zp50_admin_menu.inc as ```#include <zp50_admin_menu>```


  
  

Warning! This is an external include! It does not come bundled with AMX Mod X "out of the box" and requires additional installation. The include comes bundled with the Zombie Plague 5.0 mod for Counter-Strike 1.6 and will not work with other games (Half-Life, DoD, etc.) or without the Zombie Plague 5.0 mod installed.
