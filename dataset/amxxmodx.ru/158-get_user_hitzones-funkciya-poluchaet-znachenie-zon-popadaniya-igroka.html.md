# get_user_hitzones Функция получает значение зон попадания игрока
[The Fun Module](http://amxxmodx.ru/fun/) [fun.inc](http://amxxmodx.ru/fun/funinc/)
Так как данная функция получает значения устанавливаемые функцией **set_user_hitzones** , а я до конца ее не понял, то не хочу вас вводить в заблуждение, по этому считайте данный материал просто тестами.  
  
Инфо из **fun.inc:**  
```
/* Get user hitzones. */  
native get_user_hitzones(index, target);
```
  
  
**Синтаксис:**  

get_user_hitzones ( index, target )
  

    * **index** - id игрока   

    * **target** - Зона попадания  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <fun>  
  
#define PLUGIN "[fun.inc] set_user_hitzones"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
    register_clcmd("say /get_zone","get_zone")  
}  
public get_zone(id){  
    client_print(0,print_chat,"heat zone head :%d,%s",get_user_hitzones(id,HIT_HEAD),get_user_hitzones(id,HIT_HEAD))  
    set_user_hitzones(id,0,-100)  
    client_print(0,print_chat,"heat zone head :%d,%s",get_user_hitzones(id,HIT_HEAD),get_user_hitzones(id,HIT_HEAD))  
}
```
  
  
**Описание:**  
Надеюсь вы уже ознакомились с функцией **set_user_hitzones** , и для вас не будет ни чего нового в данном примере, так как добавился только вывод значения хит зоны до и после изменения.
