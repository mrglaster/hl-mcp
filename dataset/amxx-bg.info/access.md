# access
#### Syntax
```
stock access(id, level)
```

#### Usage
id | ```Client index```
---|---
level | ```Required admin flags```
#### Description
```
Returns if the client has the specified admin flags.
```

#### Return
```
1 if client has the admin flags, 0 otherwise
```
