et_user_footsteps
[Fun](http://127.0.0.1:8000/content/index.htm) (fun.inc) 
Description
set_user_footsteps - Gives player silent footsteps if set is 1. 
Syntax
set_user_footsteps ( id, [ set = 1 ] ) 
Type
Native 
Notes
index is a player index from 1 to 32.   
  
The set parameter can be used toggle the footsteps on or off.
