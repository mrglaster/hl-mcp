# ungag_user
#### Syntax
```
native ungag_user(name[], ip[], admin[])
```

#### Usage
name | ```Player name```
---|---
ip | ```Player IP address```
admin | ```Admin name who removed the gag```
#### Description
```
Ungags player by given name and ip
```

#### Return
```
This function has no return value.
```

#### Error
```
If the IP address is not in the valid format,
an error will be thrown
```


This code is a part of gagsystem.inc. To use this code you should include gagsystem.inc as ```#include <gagsystem>```
