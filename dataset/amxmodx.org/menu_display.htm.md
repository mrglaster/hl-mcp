enu_display
[Core](http://127.0.0.1:8000/content/index.htm) (amxmodx.inc) 
Description
menu_display - Displays a menu to a player. 
Syntax
menu_display ( id, menu, page ) 
Type
Native 
Notes
Page of the menu starts at 0. There are seven items to a page. Back/Exit/Next are automatically added as needed. You cannot use this to show a menu to everyone at once.
