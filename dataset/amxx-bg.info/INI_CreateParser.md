# INI_CreateParser
#### Syntax
```
native INIParser:INI_CreateParser();
```

#### Description
```
Creates a new INI parser.
This is used to set parse hooks.
```

#### Return
```
A new handle to an INI Parse structure.
```


This code is a part of textparse_ini.inc. To use this code you should include textparse_ini.inc as ```#include <textparse_ini>```
