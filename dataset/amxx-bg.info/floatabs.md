# floatabs
#### Syntax
```
native Float:floatabs(Float:value);
```

#### Usage
value | ```The floating point value to get the absolute value from```
---|---
#### Description
```
Returns the absolute value of a floating point value
```

#### Return
```
The absolute value
```


This code is a part of float.inc. To use this code you should include float.inc as ```#include <float>```
