# get_user_godmode
#### Syntax
```
native get_user_godmode(index);
```

#### Usage
index | ```Client index```
---|---
#### Description
```
Tells whether a player has godmode on.
```

#### Return
```
1 if player has godmode on, 0 if not
```

#### Error
```
If player is not connected or not within the range
of 1 to MaxClients.
```


This code is a part of fun.inc. To use this code you should include fun.inc as ```#include <fun>```
