# floatcos
#### Syntax
```
native Float:floatcos(Float:value, anglemode:mode=radian);
```

#### Usage
value | ```The angle to calculate the cosine from```
---|---
mode | ```What unit of measurement is the angle specified in Defaults to radians```
#### Description
```
Returns the cosine of a given angle
```

#### Note
```
For available units of measurements(modes) look at the anglemode enum
```

#### Return
```
The cosine of a given angle
```


This code is a part of float.inc. To use this code you should include float.inc as ```#include <float>```
