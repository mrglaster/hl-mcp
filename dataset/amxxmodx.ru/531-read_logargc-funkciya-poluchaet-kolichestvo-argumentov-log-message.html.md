# read_logargc Функция получает количество аргументов log message.
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
Функция read_logargc получит количество аргументов для текущего события.  
Работает только внутри форварда [**plugin_log**](http://amxxmodx.ru/core/amxmodxinc/524-plugin_log-funkciya-vyzyvaetsya-pri-vseh-log-message-sobytiyah.html)  
  
Инфо из **amxmodx.inc:**  
```
/* Returns number of log arguments.   
* Can be called only in plugin_log() forward function. */  
native read_logargc();
```
  
  
**Синтаксис:**  

read_logargc()
  
Функция возвращает число аргументов.  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <sockets>  
  
#define PLUGIN "[http://amxxmodx.ru] read_logargc"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
  
}  
  
public plugin_log(){  
      
    server_print("Log message:num arg[%d]",read_logargc())  
      
}
```
  
  
**Описание:**  
При каждом log message в консоль будет будет выведена информация о количестве аргументов.
