s_get_hostage_foll
[Cstrike](http://127.0.0.1:8000/content/index.htm) (cstrike.inc) 
Description
cs_get_hostage_foll - Returns the index of an entity of which a hostage is following. 
Syntax
cs_get_hostage_foll ( index ) 
Type
Native 
Notes
index is a hostage entity index.   
  
If 0, the hostage is not following anything.
