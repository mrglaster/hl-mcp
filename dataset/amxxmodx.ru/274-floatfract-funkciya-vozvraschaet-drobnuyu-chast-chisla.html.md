# floatfract Функция возвращает дробную часть числа.
[Ядро AMXx](http://amxxmodx.ru/core/) [float.inc](http://amxxmodx.ru/core/floatinc/)
Не думаю, что кому то из вас в ближайшее время понадобиться данная функция, хотя кто знает...  
Полезной функция наверно может быть для тех кто работает с векторами или .... если знаете, пишите в комментариях.  
  
Инфо из **float.inc:**  
```
/* Return the fractional part of a float */  
native Float:floatfract(Float:value);
```
  
  
**Синтаксис:**  

floatfract(Float:value)
  

    * **Float:value** - Дробное число.  

Функция вернет только дробную составляющую числа.  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
  
#define PLUGIN "[float.inc] floatfract"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
      
    register_plugin(PLUGIN, VERSION, AUTHOR)  
  
    new Float:fNum = 4.5  
      
    new Float:fFloat = floatfract(fNum)  
      
    server_print("Floatract: %f",fFloat)  
      
}
```
  
  
**Описание:**  
В примере создается дробная переменная и присваивается значение 4,5, в следующей строке в новую переменную выделяется дробная часть от изначального числа. Для наглядности результат выводится в консоль сервера.
