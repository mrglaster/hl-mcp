Description
dod_set_user_team - Sets player team and random class 
Syntax
dod_set_user_team ( index, teamId, [ refresh = 1 ] ) 
Type
Native 
Notes
This does not work for spectators.   
  
If refresh is set to 0, scoreboard won't be updated.
