# wl
#### Syntax
```
native wl(a, b);
```

#### Usage
a | ```Value for windows```
---|---
b | ```Value for linux```
#### Description
```
Returns a if server is windows, returns b if server is linux.
```

#### Return
```
a if windows, b if linux
```


This code is a part of okapi.inc. To use this code you should include okapi.inc as ```#include <okapi>```
