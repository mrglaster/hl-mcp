# has_all_flags
#### Syntax
```
stock has_all_flags(id, const flags[])
```

#### Usage
id | ```Client index```
---|---
flags | ```Flag string```
#### Description
```
Returns if the client has all of the specified admin flags.
```

#### Return
```
1 if the user has all of the specified flags, 0 otherwise
```


This code is a part of amxmisc.inc . To use this code you should include amxmisc.inc as ```#include <amxmisc>```
