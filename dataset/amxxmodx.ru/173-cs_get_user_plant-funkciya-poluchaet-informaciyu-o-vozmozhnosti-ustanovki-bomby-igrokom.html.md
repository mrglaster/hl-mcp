# cs_get_user_plant Функция получает информацию о возможности установки бомбы игроком.
[CStrike](http://amxxmodx.ru/cstrike/)
Если вы будите писать свой мод и у вас контр-террористы тоже смогу закладывать бомбу, а в какой то момент нет, то необходимо будет проверять, может ли игрок заминировать бомбу или нет.  
  
Инфо из **cstrike.inc:**  
```
/* Returns 1 if user has the "skill" to plant bomb, else 0. Normally this would only be true for a terrorist carrying a bomb.  
 */  
native cs_get_user_plant(index);
```
  
  
**Синтаксис:**  

cs_get_user_plant ( index )
  

    * **index** - id игрока  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <cstrike>  
  
#define PLUGIN "[cstrike.inc] cs_get_user_plant"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
  
    register_clcmd("say /plant","player_in_plant")  
  
          
}  
public player_in_plant(id){  
      
    client_print(id,print_chat,"You in plant:%s",cs_get_user_plant(id)? "Yes" : "No")  
      
    new plant = cs_get_user_plant(id)  
    client_print(id,print_chat,"You in plant:%s",plant ? "Yes" : "No")  
}  

```
  
  
**Описание:**  
Обратите внимание: Функция возвращает 1 если игрок в обще может установить бомбу, по умолчанию это должен быть террорист и у него в рюкзаке бомба.  
Поигравшись с данным примером на сервере вы легко поймете кто, что и когда может.  
Немного забегу в перед и скажу, что есть функция которая разрешит и контр террористам ставить бомбу.  
В примере показано использование функции в двух вариантах - с записью в переменную получаемого значения и напрямую.
