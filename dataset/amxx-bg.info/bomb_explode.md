# bomb_explode
#### Syntax
```
forward bomb_explode(planter, defuser);
```

#### Usage
planter | ```Planter client index```
---|---
defuser | ```Defuser client index, if applicable```
#### Description
```
Called when the bomb exploded.
```

#### Return
```
This forward ignores the return value.
```


This code is a part of csx.inc. To use this code you should include csx.inc as ```#include <csx>```


  
  

Warning: This include is compatible only with Counter-Strike 1.6 and will not function with other mods (e.g., Valve, Day of Defeat, etc.).
