od_get_pronestate
[DoDX](http://127.0.0.1:8000/content/index.htm) (dodx.inc) 
Description
dod_get_pronestate - Returns a user's prone state. 
Syntax
dod_get_pronestate ( player ) 
Type
Native 
Notes
Values are:   
  
0 (PS_NOPRONE): Not prone   
1 (PS_PRONE): Prone   
2 (PS_PRONEDEPLOY): Prone and weapon deployed
