# aemu_dac_get_reason
#### Syntax
```
native aemu_dac_get_reason();
```

#### Description
```
Gets the specified type of reason.
```

#### Note
```
: Use this native only inside the forward 'aemu_dac_query'
```

#### Return
```
type of reason
```


This code is a part of authemu.inc. To use this code you should include authemu.inc as ```#include <authemu>```


  
  

Warning! This is an external include! It does not come bundled with AMX Mod X "out of the box" and requires additional installation. Use it only if you are absolutely certain that you need it.
