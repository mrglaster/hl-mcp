[Engine](http://127.0.0.1:8000/content/index.htm) (engine.inc) 
Description
find_ent_by_model - Finds an entity in the world by class and model, returning the index. 
Syntax
find_ent_by_model ( StartEntity, Classname[], Model[] ) 
Type
Native 
Notes
If none is found, 0 is returned. If starting entity should not be specified, use -1.
