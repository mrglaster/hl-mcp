# Максимальный размер карты в HL1 
Максимальный размер карты в HL1 может быть 8196 x 8196 x 8196 единиц. Масимальный диапазон координат объектов в игре для трех осей: (-4096: +4096). Обратите внимание, что это не отрезок, а интервал, т.е. если одна из координат центра объекта будет равна или больше 4096 по модулю, то объект считается вне карты и будет либо уничтожен, либо не считаться активным (см. HLSDK, cbase.cpp, CBaseEntity::IsInWorld()).  Такие ограничения связаны со спецификой передачи координат ообъектов с сервера на клиент.  Это ограничение, правда, можно обойти, но это уже тема для мапперов, а не для кодеров.
