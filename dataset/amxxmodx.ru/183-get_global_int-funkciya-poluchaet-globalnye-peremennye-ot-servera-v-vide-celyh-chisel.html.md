# get_global_int Функция получает глобальные переменные от сервера в виде целых чисел
[The Engine Module](http://amxxmodx.ru/engine/) [engine.inc](http://amxxmodx.ru/engine/engineinc/)
Лично я в своих плагинах ни разу не использовал данную функцию, а результат ее работы получал другими функциями из других модулей, но вдруг вы захотите сделать плагин только на одно этом модуле, то вам она пригодиться.  
  
Инфо из **engine.inc:**  
```
/* Get globals from server. */  
native get_global_int(variable);
```
  
  
**Синтаксис:**  

get_global_int ( variable )
  

    * **variable** - Переменная, значение которой необходимо получить:  

      * GL_cdAudioTrack - Не знаю  

      * GL_maxClients - Максимально возможное количество игроков на сервере  

      * GL_maxEntities - Максимальное количество объектов   

      * GL_msg_entity - Не знаю  

      * GL_trace_flags - Не знаю  

      * GL_trace_hitgroup - Не знаю  

  

  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <engine>  
  
#define PLUGIN "[engine.inc] get_global_int"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
  
    server_print("Max Clients ind server: %d",get_global_int(GL_maxClients))  
  
    server_print("Max Entites: %d",get_global_int(GL_maxEntities))  
  
}  

```
  
  
**Описание:**  
Результат работы примера будет виден в консоли сервера при перезагрузке или старте.  
При инициализации плагина дважды используется функция **server_print** , в качестве подставляемого значения используется результат полученный функцией **get_global_int** , с двумя разными переменными.  
  
Если кто то из вас знает остальные значения переменных - напишите, я обязательно добавлю их описание и вас в качестве соавторов.
