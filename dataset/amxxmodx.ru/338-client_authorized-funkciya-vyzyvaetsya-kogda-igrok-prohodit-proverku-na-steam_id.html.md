# client_authorized Функция вызывается когда игрок проходит проверку на Steam_id
[Ядро AMXx](http://amxxmodx.ru/core/) [amxmodx.inc](http://amxxmodx.ru/core/amxmodxinc/)
У меня ни когда не было лицензионной копии игры, по этому я ни когда не работал с данной функцией.  
Да и тем более что данная функция вызывается между функциями [**client_connect**](http://amxxmodx.ru/core/amxmodxinc/34-client_connect.html) и **client_putinserver**.  
  
Инфо из **amxmodx.inc:**  
```
/* Called when client gets valid STEAM id (usually  
* between client_connect() and client_putinserver()). */  
forward client_authorized(id);
```
  
  
**Синтаксис:**  

client_authorized(id)
  

    * **id** - Индекс игрока (айди)  

  
  
**Тип функции:**  
Forward  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
  
  
#define PLUGIN "[amxmodx.inc] client_authorized"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
}  
public client_authorized(id){  
      
    server_print("^n^nclient_authorized^n^n")  
}  

```
  
  
**Описание:**  
Нечего тут описывать, обычный форвард вызываемый при заходе игрока.  
Для наглядности выведение текстового сообщения в консоль сервера, так как это событие наступает раньше чем игрок зайдет на сервер и не может по этому получить какое либо текстовое сообщение в чате в тот же момент когда вызывается **client_authorized**
