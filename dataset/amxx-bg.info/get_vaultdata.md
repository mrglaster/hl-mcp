# get_vaultdata
#### Syntax
```
native get_vaultdata(const key[], data[] = "", len = 0);
```

#### Description

Reads a data from given key. If len is set to zero then get_vaultdata returns value as an number. 


This code is a part of vault.inc. To use this code you should include vault.inc as ```#include <vault>```
