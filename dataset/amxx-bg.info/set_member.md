# set_member
#### Syntax
```
native set_member(const index, any:member, any:...);
```

#### Usage
index | ```Entity index```
---|---
member | ```The specified member, look at the enums with name *_Members```
#### Description
```
Sets a value to an entity's member.
```

#### Return
```
1 on success.
```
