# dod_make_deathmsg
#### Syntax
```
stock dod_make_deathmsg(killer, victim, weaponNUM)
```

#### Usage
killer | ```Killer id```
---|---
victim | ```Victim id```
weaponNUM | ```Weapon index```
#### Description
```
Sends a death message.
```

#### Return
```
This function has no return value.
```


This code is a part of message_stocks.inc. To use this code you should include message_stocks.inc as ```#include <message_stocks>```
