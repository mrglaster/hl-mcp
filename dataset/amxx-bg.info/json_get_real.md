# json_get_real
#### Syntax
```
native Float:json_get_real(const JSON:value);
```

#### Usage
value | ```JSON handle```
---|---
#### Description
```
Gets a real number.
```

#### Return
```
Real number
```

#### Error
```
If passed value is not a valid handle
```


This code is a part of json.inc. To use this code you should include json.inc as ```#include <json>```
