# okapi_get_mem_protect
#### Syntax
```
native okapi_get_mem_protect(ptr);
```

#### Usage
ptr | ```Address in memory```
---|---
#### Description
```
Retrieves the memory protection of the location pointed to by the address.
```

#### Return
```
Protection value, see PAGE_* constants
```


This code is a part of okapi_memory.inc. To use this code you should include okapi_memory.inc as ```#include <okapi_memory>```


  
  

Warning! This is an external include! It does not come bundled with AMX Mod X "out of the box" and requires additional installation. Use it only if you are absolutely certain that you need it.
