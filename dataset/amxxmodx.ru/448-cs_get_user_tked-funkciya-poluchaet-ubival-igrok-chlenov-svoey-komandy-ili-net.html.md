# cs_get_user_tked Функция получает убивал игрок членов своей команды или нет
[CStrike](http://amxxmodx.ru/cstrike/)
Предположим вы хотите наказать игрока в конце раунда если он убил союзника, то определить убивал ли игрок своего или нет поможет функция **cs_get_user_t** ked  
  
Инфо из **cstrike.inc:**  
```
/* Returns 1 of specified user has tk:ed (team killed).  
 */  
native cs_get_user_tked(index);
```
  
  
**Синтаксис:**  

cs_get_user_tked(id)
  

    * **id** - Индекс игрока  

Функция вернет 1 если игрок убил союздника  
  
**Тип функции:**  
Native  
  
**Пример:**  
```
/* Plugin generated by AMXX-Studio */  
  
#include <amxmodx>  
#include <amxmisc>  
#include <cstrike>  
  
#define PLUGIN "[http://amxxmodx.ru] cs_get_user_tked"  
#define VERSION "1.0"  
#define AUTHOR "Admin"  
  
  
public plugin_init() {  
    register_plugin(PLUGIN, VERSION, AUTHOR)  
      
    register_clcmd("say tk","test_tk")  
}  
public test_tk(id){  
  
    client_print(0,print_chat,"Have tk? %s ", cs_get_user_tked(id) ? "Yes" : "No")  
      
}
```
  
  
**Описание:**  
Выполнив команду **say tk** вы получите в чат ответ - убивал ли вы союзников или нет.
